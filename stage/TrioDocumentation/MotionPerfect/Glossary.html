Glossary

EtherCAT

A communications bus system developed by Beckhoff, and adopted as as an international standard to control industrial devices including servo drives. It uses a special communications protocol (specified in IEC 61158) over a daisy chained Ethernet 100BASE-TX dedicated network.

FEC

Feature Enable Code: A code which has to be entered into the controller before a software enablable feature can be used. An FEC normally has to be purchased and is unique to one controller.

GTA

Global Target: A TARGET point, globally defined for the Robotics Programming System.

HMI

Human Machine Interface: An interactive device used to control a machine and display machine parameters and status. Trio’s Uniplay device is an example of one of these devices.

MC

Motion Coordinator:  A Trio motion controller.

RPS

Robotic Programming System: The system used to program the motion of a robot in Motion Perfect.  Documentation can be found in the TrioBASIC help file.

RTEX

A communications bus system developed by Panasonic to control some of their servo drives. It uses a special communications protocol over a daisy chained closed loop Ethernet 100BASE-TX dedicated network. 

SCARA

Selective Compliance Articulated Robot Arm: A type of robot arm developed in Japan for assembly applications. The arm is rigid in the Z-axis and pliable in the XY-axes, which allows it to adapt to holes in the XY-axes.

Uniplay

Trio’s HMI system.

VR

Global Variable: An array variable available to all Trio BASIC programs as well as the Uniplay system.

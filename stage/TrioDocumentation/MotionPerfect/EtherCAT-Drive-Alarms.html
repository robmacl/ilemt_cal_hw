EtherCAT Drive Alarms

The Alarms page shows details on the drive status.  It is split into two panels, the top panel shows the active alarm and the bottom panel shows a history of previous alarms.

Active Alarm

If there are no active alarms this panel will show the text ‘No active alarm’. 

If there is an alarm present on the drive this panel will show the alarm code, the name and the description.  Below the active alarm is an extended description with some trouble shooting tips on how to resolve the source of the error and a button to clear the error (‘Clear Active’). Pressing this button will attempt to clear the error on the drive only.  Any assosiated error on the controller will not be cleared using this button.

Alarm History

The Alarm History shows a list of previous alarms in chronological order with the most recent at the top.  The ‘Clear History’ button will remove all entries from the error history.

Motor Mismatched

The parameter data and encoder data describing the motor are inconsistent

For example, using a 400W motor and a 750W drive

In some circumstances it may be possible to use a larger drive to run a smaller motor.

Contact Trio or the Authorized Distributor.

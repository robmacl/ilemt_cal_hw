Controller Tools

Motion Perfect 3 has several tools which are used to configure the controller and interact with it. Most of these tools are available from the "Controller" section of the Main Menu .

Tool

Description

Connection Settings

Settings for the communications interface on the PC used by Motion Perfect to communicate with the controller.

Reset Controller

Performs a soft reset on the controller.

Communications

Settings for the communications interfaces on the controller.

Enable Features

Enable or disable software configurable features on the controller.

Memory Card

Manipulate files stored on the memory card in the controller.

Load Firmware

Load system firmware onto the controller.

Reprogram FPGA

Reprogram the FPGA on the controller (if supported).

Directory

Show a full directory listing of the programs on the controller.

Processes

Show details of the processes currently running on the controller.

Import / Export Values

Import or export ranges of TABLE or VR values.

Lock / Unlock Controller

Lock or unlock the controller.

Date And Time

View or change the real-time clock on the controller.

Package Authoring Example

Below is an example of a package development project for the “MyTestPackage” package. All the items of the future package are placed in “MyTestPackage” group.

The “MyTestPackage” group (and the future package) contains two BASIC libraries and one HMI library with some HMI content and images.

A package can be created out of this group using the context menu command “Create Package From Group...”:

This will bring up the package creation wizard which will guide the user through the process of package creation. It has a several pages with the following content:

We do not limit the package by firmware version, but specify controller types that the package is relevant to.

 

 

We allow the package to be freely loaded and displayed by the HMI simulator (which does not apply to physical Uniplay devices!). We also specify the HMI components/properties that will be available to package users.

Clicking on “OK” creates the package and installs it on the local system.

Programs

Programs are items in a Motion Perfect project which can be executed (run) on a Trio Motion Coordinator (Controller) and functional items which can be loaded onto a Uniplay HMI device using a Trio Motion Coordinator. Motion Perfect handles programs as part of a project .

Program Types

Motion Perfect supports several different program types as follows:

Icon

Type

Note

TrioBASIC

A TrioBASIC program

TrioBASIC Library

A TrioBASIC function library

MC_CONFIG

A Controller Configuration program

Encrypted TrioBASIC

This type of file can only be written to a controller, it cannot be read. It is produced by encrypting an normal TrioBASIC program.

Text

This is textural information stored on the controller and does not represent a runnable program.

IEC Task

Consists of one or more of the IEC program types below.

IEC Library

A library which can contain one or more of the IEC program types below.

IEC Ladder Diagram

IEC Ladder program

IEC Structured Text

IEC Structured Text program

IEC Function Block Diagram

IEC Function Block Diagram program

IEC Sequential Function Chart

IEC Sequential Function Chart program

HMI Design

A design for a Uniplay HMI (contains one or more HMI pages).

HMI Page

A page used in an HMI design

HMI Image

An image used in an HMI page

Cam Library

A library to contain one or more cam profiles. Only a single cam library is allowed which must be named “CAMLIB”.

Cam Profile

A cam profile specified as set of TABLE values with or without associated TrioBASIC code. The profiles are designed for inclusion in TrioBASIC programs.

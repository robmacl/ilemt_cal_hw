Pre-installed Templates

Motion Perfect comes with a set of templates. Some of them are starters for known paradigms while others are samples, from which one can learn and experiment with. A short description is available and displayed along with the template name. Usually the template project will contain a more detailed description in a text format (file).

It is possible to get up-to-date templates (update) from the Trio Motion Technology update site straight from the Project Template Manager tool in Motion Perfect.

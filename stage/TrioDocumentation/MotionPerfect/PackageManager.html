Package Manager

The package manager tool manages the repository of packages which are currently available to Motion Perfect, both those installed during Motion Perfect installation and those installed by the user. It can be started using “Project”→”Packages”→”Package Manager...” from Motion Perfect’s main menu.

 

The package manager window when started:

 

Section

Description

Show

All

Displays all the available packages.

Standard

Displays only the packages that come with Motion Perfect installation.

User

Displays only the packages that are manually installed/created by the user.

 

Filter by tag

Contains all the tags defined in the packages. Allows the user to filter out packages by tag.

Packages

Package list

Versions available

List of versions of the selected package.

 

Button

Operation

Add To Project

Adds the selected package to the current project.

See Add Package To a Project .

Update

Check for online updates or update from a file.

See Updating Packages .

Import

Imports a user package from a package file.

Share

Exports the selected user package to a file or to an email. This function does not work for installed packages. The package user(recipient) will need a project key to load (add) this package to a project. Such a key can be generated through the standard project key generation routine (“Project”→”Manage Encryption…”)

Delete

Deletes the selected user package. This function does not work for installed packages.

Close

Closes the package manager window.

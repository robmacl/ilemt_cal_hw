MC_CONFIG File

The MC_CONFIG File is an initialisation script which can be used in Series 4 and later Motion Coordinators. The MC_CONFIG File, if present, is always run automatically at power-up. Although the MC_CONFIG file looks like a normal Trio BASIC program it can only contain a very limited number of parameter assignments.

Using the MC_CONFIG file is the recommended way to set system parameters. Any system parameters which are stored in flash memory on the controller and are not specified in the MC_CONFIG file will retain the value stared in flash memory. This also applies if there is no MC_CONFIG file.

Changes made to the MC_CONFIG file are only applied to the controller when it is reset. If changes are made to the MC_CONFIG file, the controller will not run any other programs until it has been reset.

Details of the parameters and commands can be found in the Trio BASIC command reference.

The list of settable parameters depends on the controller type, the firmware version and possibly the controller configuration. The above list should only be considered as a guide as to the parameters available.

Editing the MC_CONFIG file is done using a dedicated MC_CONFIG editor .

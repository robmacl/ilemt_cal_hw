EtherCAT Device Info

The EtherCAT information page is displayed differently depending on the type of device.  There are currently 4 different device types:

Trio Drive

Trio Device

Third Party Drive

Third Party Device

 

Trio Drive

Screen shot of EtherCAT Device screen showing a DX4 servo drive

 

EtherCAT Info

Position    The position of device based on Network Topology.  EtherCAT will number devices logically based on the sequence that they physically wired together.

Alias         Device identifier used in preference to the network position for addressing the device.  This value is stored locally on the device

Address    Configured station alias

State        EtherCAT State; Init, Pre-Op, Safe-Op, Operational

Device Info

            Vendor ID        EtherCAT vendor ID ($000002DE for Trio Motion Technology)

Vendor            Name of device vendor

Product Code EtherCAT product code

Model              Device model description

Revision           Revision of EtherCAT slave information (ESI) file

Serial Number             Device serial number

Hardware Version      Device hardware version number

Firmware Version       Device firmware version number

FPGA Version              Device FPGA version number

Load Firmware…        Update firmware image on device by downloading file (more information on this can be found at Update firmware )

Reprogram FPGA…     Update FPGA image on device by downloading file (more information on this can be found at Update firmware )

State

Status              rdY, run, diS, SAF, FLt, etc…

Main Power    Indication of the state of the mains power, either ON or OFF.  The actual voltage can be monitored from the Monitor page.

Alarm              Alarm code (more information on this can be found at Alarms )

Overtravel      P-OT, N-OT, P/N OT

Advanced        Indication of state of advanced mode, either ON or OFF.  Advanced mode requires entry of an unlock password.

FoE

Allows generic file upload and download to EtherCAT devices.  N.B. Not all EtherCAT devices support generic file upload and/or download

Download File…     Download a file from the PC to the device via File over EtherCAT (FoE) protocol

Upload File…          Upload a file from the device to the PC via File over EtherCAT (FoE) protocol

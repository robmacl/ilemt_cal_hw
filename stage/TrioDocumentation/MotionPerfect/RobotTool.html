Robot Tool

The Robot Tool provides the ability to display and modify various aspects of controller configuration related to kinematic groups.

Important: Before starting the robot tool, please make sure your Motion Coordinator is set up for robot support. Robot settings are done from robot definitions tool which will generate/change ROBOT_DEFINITIONS program. The latter needs to be run once to set-up your Motion Coordinator for robot support.

 

The tool generates/modifies the following programs:

-           ROBOT_TOOL_AND_FRAMES – contains set commands for the defined tool offsets as well as object, robot frames and collision objects.

-           ROBOT_GLOBAL_TARGETS – contains set commands for the defined GTA objects.

It is not recommended to edit those programs outside the Robot Tool.

The user interface consists of several pages representing detailed information for the following items:

-           global targets (GTAs)

-           tool offsets

-          object frames

-           collision objects

-           robot frames

 

Pressing the “Save to programs” button at any time will save the data from the tool to the corresponding programs. This will also be done automatically when the tool is closed.

After the ROBOT_DEFINITIONS program has been run once a following execution of ROBOT_TOOL_AND_FRAMES and ROBOT_GLOBAL_TARGETS will accomplish your robot set-up.

For robot definition and parametrization see (ROBOT_DEFINITIONS) see Robot Definitions reference.

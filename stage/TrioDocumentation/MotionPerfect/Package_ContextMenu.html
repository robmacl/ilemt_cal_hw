Package Context Menu

Package management commands are available from the package item context menu. It also provides a link to the package documentation.

Menu Entry

Operation

Stop All (HALT)

Stops all the running programs from that package.

Remove

Removes the package from the project.

Update Package

Runs the package update procedure. This will suggest and optionally update/downgrade the package to a newer/older version if available.

View help file

Opens the help file provided with the package (if any).

Properties

Displays the package properties.

It is expected that the package contains BASIC libraries or programs, or HMI libraries which will help you build your project. A documentation file should be available with each package should contain a definition of the package interface, help on provided functions, help on exposed HMI content, examples, etc.

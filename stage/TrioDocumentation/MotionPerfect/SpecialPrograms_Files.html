Special Programs & Files

The controller uses two special programs/files for configuration purposes:

MC_CONFIG File

This is run before any programs are run. It’s content is restricted to a limited number of Trio BASIC commands which are used to configure system parameters. This file, if present, is always automatically run at power-up.

STARTUP Program

This is used to configure data (VR variable and TABLE values), Axis Parameter values, and some communications parameters. The STARTUP program is not run automatically and must be scheduled to run by the user.

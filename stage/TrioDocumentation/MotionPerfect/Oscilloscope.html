Oscilloscope

The software oscilloscope can be used to trace many of the variables and parameters on the Motion Coordinator, aiding program development and machine commissioning.

There are up to 8 channels available, each capable of recording at up to 1000 samples/sec, with manual cycling or program linked triggering.

The controller records the data at the selected frequency, and then uploads the information to the oscilloscope to be displayed. If a larger time base value is used, the data is retrieved in sections, and the trace is seen to be plotted in sections across the display. Exactly when the controller starts to record the required data depends upon whether it is in manual or program trigger mode. In program mode, it starts to record data when it encounters a TRIGGER instruction in a program running on the controller. However, in manual mode it starts recording data immediately.

Oscilloscope Main Menu

There are 4 sub-menus available:

  1. Traces
  2. Cursors
  3. View
  4. Profiles
  5. Settings

Traces

Entry

Description

Add New Trace

Adds a new trace to the end of the existing trace list, up to a maximum of 8

Load

Load a trace (selected from a list)

Save

Save a trace (selected from a list)

Reset

Resets a trace (selected from a list). All values returned to default and the trace turned off

Remove

Remove a trace (selected from a list)

Export Traces

Exports trace data to a file. The user can select which traces are stored.

Cursors

Entry

Description

Add New Cursor

Adds a new cursor to the end of the existing cursor list up to a maximum of 8

Map

Maps a cursor onto a trace (cursor then trace selected from lists)

Reset

Resets the cursor position to zero (cursor selected from list)

Remove

Removes (hides) a cursor (selected from list)

Go To

Got to cursor (selected from list). This is used when zoomed in and it attempts to scroll the current view until the selected cursor in the centre

Centre in View

Positions the cursor (selected from a list) to the centre of the current view.

View

Entry

Description

Take Snapshot

Takes a snapshot of the oscilloscope trace area and copies it to the Windows Clipboard

Profiles

Entry

Description

Add New Profile

Adds a new oscilloscope profile with trace and cursor configuration reset to defaults

Profile list

For each defined profile there is a corresponding menu entry allowing to select the profile as well as to rename, copy or delete it.

Settings

Entry

Description

Export Settings

Export the current settings to a file

Import Settings

Load a previously exported set of settings from a file

Reset Settings

Resets all the settings (including trace and cursor settings) to their default values

Settings

Allows to user to change the oscilloscope settings

Controls

There are four groups of controls, one for each of the oscilloscope's four channels, a group of horizontal function controls and a group to control up to four cursors.

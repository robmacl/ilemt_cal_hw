Simulator Manager

The Simulator Manager tool is used to control the running of multiple instances of the MV400 controller Simulator.

Clicking on “Run New Simulator” will start a running instance of the simulator on the next available IP Address in the 127.0.0.x range.

Each simulator entry has four buttons to control that simulator.

Button

Function

Run (start) simulator

Stop simulator

Edit simulator properties

Stop simulator (if running) and remove from list

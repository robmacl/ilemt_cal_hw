Main Menu

The Main Menu has a set of sub-menus which splits the menu commands into functional groups as follows

Project

New

Create a new project and erase any controller content

Load

Load an existing project onto the controller

Change

Change to a different project and reconcile with the existing controller contents

Create from Controller

Create a new project from the existing controller contents

Save

Save the current project (flushes all changes to disk)

Save As

Save the current project under a different name

Export

Export the project in a different format

Manage Encryption

Project encryption and encryption key generation

Project Check

Check the current project against the controller contents

Backups

Open the " Backup Manager " tool to create or manage project backups

Close

Close the current project (this results in the connection changing to Tool Mode)

Import File/Program…

Load an existing file or program and add to the current project.

Modify STARTUP program

Modifies the STARTUP program

Recent Projects

Allows easy working with recently used projects

Solution Manager

Opens the solution manager to allow working with more than one controller

Print

Prints the current active editing session

Exit

Exits from the application

Controller

Connect in Sync Mode

Connect to the controller in Sync Mode

Connect in Tool Mode

Connect to the controller in Tool Mode

Connect In Direct Mode

Connect to the controller in Direct Mode

Disconnect

Disconnect from the controller

Connection Settings

Change the connections settings used for the communicating with the controller

Reset Controller

Reset the controller by performing a warm restart

Communications

Open the sub-menu which allows the configuration of all communications interfaces on the controller.

Enable Features

Enable and disable soft features

Memory Card

Open the " Memory Card Manager " to manipulate the contents of the memory card in the controller.

Load Firmware

Load new system firmware

Reprogram FPGA

Reprogram the FPGA on the controller.

Not all controllers support this function.

Directory

Shows an extended directory listing of the programs on the controller

Processes

Shows a list of all user processes currently running on the controller

Import Values

Import TABLE and VR values form a file and write to controller.

Export Values

Read TABLE and VR values from the controller and save in a file.

Download All Files

Download all the files on the controller.

Lock Controller

Lock the controller using a locking code

Unlock Controller

Unlock a locked controller

Date and Time

Sets the real-time clock on the controller using the " Date and Time " tool

HMI

HMI (Uniplay) management functions

Edit

Undo

Undo the last editing operation

Redo

Redo the last undone editing operation

Cut

Cut the currently selected text into the clipboard

Copy

Copy the currently selected text into the clipboard

Paste

Paste test from the clipboard

Select All

Select all text in the document

Select None

Deselect the current selection

Delete

Delete the currently selected text

Trio BASIC

Open the Trio BASIC sub-menu which gives access to reformatting and auto-commenting operations.

HMI

Editing functions for controls in the HMI designer

Search

All search commands apply to the current active editing session

Find

Search for a text string

Find Next

Find the next occurrence of the last search string

Find Prev

Find the previous occurrence of the last search string

Find Next Occurrence Current Selection

Find the next occurrence of the currently selected text string

Find Prev Occurrence Current Selection

Find the previous occurrence of the currently selected text string

Replace

Replace one text string with another

Find in Project

Find a text string in all files in the current project

Replace in Project

Replace a text string in all files in the current project

Toggle Bookmark

Toggle a bookmark on the current line

Goto Next Bookmark

Go to the next bookmark

Goto Prev Bookmark

Go to the previous bookmark

Goto Line/Label

Go to a line or label

Match Scope

Go to the end / beginning of the scope started / ended on the current line

File/Program

New

Create a new empty program (see " Creating a New Program ")

Load

Load an existing program and add to the current project

Save to Disk

Save a program from the current project to a disk file.

Debug

Debug a program in the current project

Save

Saves any changes to disk

Copy

Copy a program in the current project

Rename

Rename a program in the current project

Delete

Delete a program in the current project

Delete All

Delete all programs in the current project

Compile All

Compile all programs in the current project

Set AUTORUN

Set the Autorun process of a program in the current project

Run AUTORUN programs

Run all programs set to auto-run

Stop All (Halt)

Stop all running programs

Build/Run

The commands I n this sub-menu operate on the program open in the current active editing session.

Compile

Compile the program (any changes are saved first)

Run

Run the program

Step

Step the program

Step In

Step program into a function or subroutine

Step Out

Step program out of a function or subroutine

Pause

Pause program execution

Stop

Stop program execution

Toggle Breakpoint

Toggle breakpoint on the current line

Enable/Disable Breakpoint

Toggles the enabled state of the breakpoint on the current line

Breakpoints

Opens a dialog to display all current breakpoints

Watch Variable

Add a watch for the currently selected variable

Set AUTORUN

Set the AUTORUN process number

The availability of the commands in the Build/Run sub-menu depends on the type or program being edited and the run state of the program.

Tools

Axis Parameters

View and modify axis parameters using the " Axis Parameters " tool.

Intelligent Drives

Configure intelligent drives attached to the controller. This is implemented using add-ons.

Oscilloscope

A software Oscilloscope tool which can be used to show traces of how parameters vary with time.

Digital I/O Status

View the states of digital inputs and outputs and change the state of digital outputs using the " Digital I/O Viewer " tool.

Analogue I/O Status

View the states of analogue inputs and outputs and adjust analogue outputs using the " Analogue I/O Viewer " tool.

CamGen

Generate and edit cam profiles. This is implemented as an add-on. This add-on has its own help file.

3D Visualization

Visualize the motion of an machine controlled by the connected Trio controller. This add-on has its own help file.

Robot Tool

 

Jog Robot

 

Jog Axes

Manually jog axis positions using the " Jog Axes " tool.

Table Viewer

View and change table data values using the " Table Viewer " tool.

VR viewer

View and change VR variable data values using the " VR Viewer " tool.

Watch Variables

View and change the values of local and global variables whilst debugging using the " Variable Watch " tool.

Setup Uniplay HMI device

Configure a Uniplay HMI device.

Show HMI Events Log Window

Displays the HMI events log.

Terminal

Open a Terminal Tool to interact with the controller.

Manage Simulators

Configure running instances for the MC400 controller simulator.

Diagnostics

Configure Diagnostics for fault finding.

Options

Change the Options for Motion Perfect and its tools.

Motion Perfect Factory Reset

Clears all Motion Perfect’s settings, caches etc. so that it is in a similar state to a freshly installed Motion Perfect on a new PC.

Window

Toolbar

Show / hide the main toolbar

Status Bar

Show hide the application status bar

Output Window

Show / hide the Output Window

Controller Tree Window

Show / hide the Controller Tree Window

Project Tree Window

Show / hide the Project Tree Window

Toolbox

Show / hide the Toolbox

Show Recent Work

Show the Recent Work dialog

Clear Output Window

Clear the Output Window

Close Window

Close the current window

Close All Documents

Close all open editing sessions

Reset Window Layout

Reset the window layout to the default layout

Help

Motion Perfect v5.0 Help

Displays Motion Perfect help

TrioBASIC Help

Displays TrioBASIC language help

IEC 61131-3 Help

Displays help for IEC 61131-3 programming and debugging using Motion Perfect

Standard IEC 61131-3 Language and Function Help

Displays the reference for the IEC 61131-3 languages and functions implemented on the Motion Coordinator

Trio IEC 61131-3 Function Blocks Help

Displays help for Trio specific IEC 61131-3 function blocks.

Trio PLCopen Function Blocks Help

Displays help for PLCopen function blocks.

CamGen Help

Displays help for the CamGen tool.

3D Visualizer Help

Displays help for the 3D Visualizer tool.

HMI Help

Displays help for the HMI designer and Uniplay device support.

Check for Updates

Check for newer versions of Motion Perfect (Internet connection required).

Report a Problem

Generates a debug report and emails it to Trio. (Requires a working email account and an Internet connection).

Contact Technical Support…

Opens a web browser showing the Technical Support page on Trio’s main website (Internet connection required).

Show Release Notes…

Displays the release notes for this version of Motion Perfect.

About Motion Perfect v5.0…

Displays the Motion Perfect About Box which shows software versions.

Robot Project Context Menu

When present the robot project item has its own context menu in the Controller Tree.

 

 

Operation

Notes

New

Create a new empty robot program and add it to the robot project

Load…

Load a robot program from disk and add it to the robot project

Compile All Robot Programs

Compile all robot programs

Stop All (Halt)

Stop all running robot programs

Load Robot Project

Loads a robot project and contained robot programs from a folder on disk and replaces the current robot project on the controller

Save Robot Project

Saves the robot project and all contained robot programs to a folder on disk

Rename Robot Project

Rename the robot project

Delete All Robot Programs

Delete all robot programs from the robot project

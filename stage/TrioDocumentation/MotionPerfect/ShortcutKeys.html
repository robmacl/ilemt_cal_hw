HMI editor

These shortcut keys require an opened HMI editor that has the input focus.

Shortcut

Command

Arrow key

Moves the selected items

Alt + arrow key

Resizes the selected items

Ctr + mouse drag

Creates a copy of the dragged item

Alt + mouse drag

Moves the selected items without object snapping (when object snapping is toggled ON from the tool bar)

Ctrl+A

Select all

Del

Delete selection

Ctrl + mouse click

Adds/removes the item from the current selection

Alt + mouse lasso selection

Removes the surrounded items from the selection

Ctrl + mouse lasso selection

Adds the surrounded items to the selection

F7

Performs HMI validation

F8

Save and upload the design (if modified)

Ctrl+Z

Undo. (Some actions cannot be undone)

Ctrl+Y

Re-do

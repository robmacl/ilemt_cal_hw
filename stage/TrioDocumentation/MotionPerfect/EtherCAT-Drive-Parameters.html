The parameter screen allows advanced access to all drive parameters.  The parameters are shown in a tabulated form, grouped into logical sets.

The icons across the top of the page allow control of the table view along with some additional functionality.

Icon

Function

Force read of selected parameter from drive

Force read of all parameters from the drive

Force write of any un-written parameters to the drive

Reset all parameters to factory default

Import a parameter set from a file

Export a parameter set and save to a file or create a STARTUP program

Expand / Collapse sub-groups

Text based search function, allow easy search of parameter table for specific items

Enable / Disable parameter filters

Expand all groups

Collapse all groups

Each parameter entry shows:

  • parameter number
  • name
  • an indication of if a change would require the drive to be restarted
  • the current value
  • the default value
  • the allowable range

The parameter values shown in the parameter table are read from the drive when the screen is opened so will always be up-to-date.

When a value is changed it will automatically be written to the drive.

If the current value is not the default, then the default value will be shown with a pink background.

When the value is changed it will appear with a green background to indicate the change until it has been successfully been applied to the drive.

If the change requires the drive to be restarted, this will be indicated in the drive menu by hightlighting the restart button.

At the bottom of the table are quick filters which allow one-click application of filters.

At the bottom of the page is a parameter descripton which will show a description of the selected parameter.

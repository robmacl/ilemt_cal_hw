FWD_JOG

Type:

Axis Parameter

Description:

This parameter holds the input number to be used as a jog forward input.

When the FWD_JOG input is active the axis moves forward at JOGSPEED .

The input used for FWD_JOG is active low.

It is advisable to use INVERT_IN on the input for FWD_JOG so that 0V at the input disables the jog.

FWD_JOG overrides REV_JOG if both are assigned to the same input and activated. If they are assigned to different inputs and activated one after the other, the first command takes precedence, the second command is ignored.

Value:

-1

Disable the input as FWD_JOG (default)

>0

Input number as FWD_JOG

Example:

Initialise the FWD_JOG so that it is active high on input 7

INVERT_IN(7,ON)

FWD_JOG=7

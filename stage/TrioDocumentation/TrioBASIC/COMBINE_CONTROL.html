COMBINE_CONTROL

Type:

Axis Command

Syntax:

COMBINE_CONTROL(mode, source_1, source_2, Numerator_1, Denominator_1, Numerator_2, Denominator_2)

Description:

The COMBINE_CONTROL command is used to set up the mode of operation for the COMBINE_AXES function.

Parameters:

mode:

0

DPOS out = source_1 position + source_2 position

1

DPOS out = source_1 position – source_2 position

2

√ (source_1 MPOS ^2 + source_2 MPOS ^2)  [1]

3

DPOS out = (source_1 position + source_2 position) / 2

source_1

0 = DPOS, 1 = MPOS

source_2

0 = DPOS, 1 = MPOS

numerator_1

Applied to source_1 before the value is used in the function

denominator_1

Applied to source_1 before the value is used in the function

numerator_2

Applied to source_2 before the value is used in the function

denominator_2

Applied to source_2 before the value is used in the function

[1] Mode 2 has only 3 parameters:

COMBINE_CONTROL(mode, offset_1, offset_2)

mode

2

offset_1

Offset applied to source_1 value

offest_2

Offset applied to source_2 value

 

Examples:

Example 1:

Sub routine to set up an axis to have position defined by the sum of 2 other axes.  Set the first source axis to have 0.5 multiplier and the second source to be x2.

DPOS out = MPOS1 * 0.5 + MPOS2 * 2

'=============================

' source1 + source2

mode_0:

source1 = 10

source2 = 11

out_axis = 12

 

source_mpos = 1

source_dpos = 0

 

'COMBINE_CONTROL(mode, source  1, source  2, Numerator  1, Denominator  1, Numerator  2, Denominator  2)

COMBINE_CONTROL(0, source_mpos, source_mpos, 1, 2, 500, 250)

COMBINE_AXES(source1, source2)

BASE(source1, source2)

MOVE(100, 120)

WAIT IDLE

 

PRINT DPOS AXIS(out_axis)

 

RETURN

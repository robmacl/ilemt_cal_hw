VFF_GAIN

Type:

Axis Parameter

Description:

Servo axes and remote speed (CSV) axes; The velocity feed forward gain is a constant which is multiplied by the change in demand position. Velocity feed forward gain can be used to decreases the following error during constant speed by increasing the output proportionally with the speed. For a velocity feed forward  Kvff and change in position ΔΡd, the contribution to the output signal is:

0vff = Kvff x ΔΡd

Remote position axes like EtherCAT CSP and RTEX position axes, the VFF_GAIN applies a target position (P t) advance that is proportional to DEMAND_SPEED.

P tout = P t + ΔΡd x K vff

Value:

Velocity feed forward constant (default =0)

Example:

Set the VFF_GAIN on axis 15 to 12

BASE(15)

VFF_GAIN=12

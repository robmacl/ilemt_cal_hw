STATIC.. AS..

Type:

Program Variable

Syntax:

STATIC name AS BOOLEAN

STATIC name AS FLOAT

STATIC name AS INTEGER

STATIC name AS STRING

STATIC name AS BOOLEAN[(size1 [,size2 [,size3]])]

STATIC name AS FLOAT[(size1 [,size2 [,size3]])]

STATIC name AS INTEGER[(size1 [,size2 [,size3]])]

STATIC name AS STRING([size1 [,size2 ,]]length)

Description

Keyword STATIC (which is like Visual Basic STATIC keyword) has been added to allow the option of retaining the value of a function variable when it is called repeatedly, i.e., the variable will only be initialised the first time that the function is called.

The following code snippet illustrates how different classes of variable are initialised:

FUNCTION myfunc()

  DIM f1 AS FLOAT 'Float variable will re-initialise to 0.0 each call

  STATIC f2 AS FLOAT 'Float variable will retain its value

  DIM array1 AS INTEGER(5,10) 'Array will re-initialise to 0 each call

  STATIC array2 AS INTEGER(100) 'Array will retain its value

...

ENDFUNC

Use in Function Library only

Variable types:

BOOLEAN

1-bit binary value ( TRUE or FALSE )

FLOAT

64-bit floating point number (default)

INTEGER

64-bit signed integer value

STRING

ASCII text

Examples:

Use a local variable as a flag to track the ok status of a machine.

FUNCTION mysum(ival AS INTEGER) AS INTEGER

    STATIC total AS INTEGER = 1000 'total will start at 1000 and then add ival each call

 

    total = total + ival

 

    RETURN total

ENDFUNC

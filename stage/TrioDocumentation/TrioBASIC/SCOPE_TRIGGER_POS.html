SCOPE_TRIGGER_POS

Type:

System Parameter

Syntax:

value = SCOPE_TRIGGER_POS

Description:

SCOPE_TRIGGER_POS returns the index position of the trigger point when SCOPE_DELAY has been set up for pre-trigger or delayed trigger.

If SCOPE_DELAY is a negative value, then pre-trigger occurs. If there are fewer points captured than the SCOPE_DELAY is set for, use SCOPE_TRIGGER_POS to find the actual trigger index in the TABLE.

Value:

The TABLE index where the most recent scope trigger point occurred.

Examples:

Example 1:

Set Scope to pre-trigger at 500 ms. Print a message with scope completed and the trigger position to terminal #5.

  'capture 1000 points on 2 channels
  'pre-trigger to capture 500 points before the TRIGGER
  'and 500 after TRIGGER
  SCOPE_DELAY = - 500
  SCOPE ( ON , 1 , 10001 , 12000 , MPOS AXIS ( 0 ), MPOS AXIS ( 1 ))
  WA ( 320 )
  TRIGGER
  WAIT UNTIL SCOPE_POS = 1000
  PRINT # 5 , "Scope capture complete"
  PRINT # 5 , "Actual Trigger at TABLE " ; 10000 + SCOPE_TRIGGER_POS

This deliberately does not leave enough time to capture 500 points.  It is to show the use of SCOPE_TRIGGER_POS.

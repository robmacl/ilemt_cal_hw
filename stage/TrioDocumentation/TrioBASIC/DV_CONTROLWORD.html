DV_CONTROLWORD

Type:

System Command

Syntax:

value = DV_CONTROLWORD(channel)

DV_CONTROLWORD(channel) = value

Description:

Sets up the operation of a remote EtherCAT Flexslice device such as the thermocouple module or RTD module.

Parameters:

value:

The value read from or written to the device variable’s control word

channel:

Channel number

Examples:

See Also:

CONTINUE

Type:

Program Structure

Syntax:

CONTINUE

Description:

This command applies to the loop structures WHILE…WEND, FOR TO…NEXT and REPEAT…UNTIL.

Once this command is executed within one of the structures mentioned above, the execution will jump directly to the next iteration, therefore the code below this instruction will not be executed during the current iteration of the loop.

Example:

While input 12 is off, move the base axis and flash an LED on output 10, if input 9 is active, the loop will jump to the next iteration after the base axis moves 200.

WHILE IN(12) = OFF

  MOVE(200)

  WAIT IDLE

  IF IN(9) = ON THEN

    CONTINUE

  ENDIF 

  OP(10, OFF)

  MOVE(-200)

  WAIT IDLE

  OP(10, ON)

WEND

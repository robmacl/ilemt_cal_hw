SET_MOTIONSP

Type:

System Function

Syntax:

SET_MOTIONSP(axis, force_speed[, startmove_speed[, endmove_speed[, force_accel[, force_decel[, force_jerk[, force_ramp[, force_dwell]]]]]]])

Description:

SET_MOTIONSP is used to pre-set the parameter list for a subsequent MOVESP, MOVEABSSP, MOVECIRCSP etc. using just one command.

This command requires the axis to be specified as a parameter and therefore does not use the current base axis, nor will it accept the AXIS modifier.

To not use a parameter, that parameter’s default value must be specified, which for speed, acceleration and jerk is -1, and for a ramp or dwell is zero.

Parameters:

Minimum of the first 2 parameters must be set.  All other parameters are optional.

axis:

Axis number to be set up

force_speed:

The value for FORCE_SPEED to be set

startmove_speed:

The required value for STARTMOVE_SPEED

endmove_speed:

The required value for ENDMOVE_SPEED

force_accel:

The required value for FORCE_ACCEL

force_decel:

The required value for FORCE_DECEL

force_jerk:

The required value for FORCE_JERK

force_ramp:

The required value for FORCE_RAMP

force_dwell:

The required value for FORCE_DWELL

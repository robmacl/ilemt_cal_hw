PI

Type:

Constant

Description:

The number π is the constant equal to the ratio of a circle’s circumference to its diameter, which is approximately 3.14159.

Examples:

Example 1:

To print the radius of a circle of given circumference.

circum = 100

PRINT "Radius = "; circum / (2 * PI)

Example 2:

Set the axis calibration to work in user UNITS of Radians.

'Motor has 8192 counts per turn.

UNITS = 8192 / (2 * PI)

ROBOT_IO

Type:

Robotics Parameter (Read Only)

Syntax:

ROBOT_IO(group)

Description:

Returns the states of the kinematic group’s digital I/O as a bit pattern.

The least significant 16 bits contain the kernel’s digital input states. The most significant 16 bits contain the kernel’s digital output states.

ROBOT_IO requires the RPS runtime FEC

RPS 2.0 kernel must be activated using the RPS_KERNEL_V2 keyword to enable its use.

Examples:

Check if Robot(0) is enabled.

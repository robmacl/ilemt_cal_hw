SYSTEM_WARNING

Type:

System Parameter

Description:

SYSTEM_WARNING is a bitwise system variable that returns a value which indicates which active warnings have been reported by the firmware.

Value:

RFID

0x000001

EBUS overcurrent

0x000002

USB1 overcurrent

0x000004

USB2 overcurrent

0x000008

Power supply undervoltage

0x000010

PCMCAT Windows RAM low

0x000020

MotionPLC slice output driver fault

0x000040

MotionPLC built-in output driver fault

0x000080

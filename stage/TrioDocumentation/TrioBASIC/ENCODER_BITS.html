ENCODER_BITS

Type:

Axis Parameter (MC_CONFIG)

Description:

This parameter is only used with an absolute encoder axis. It is used to set the number of data bits to be clocked out of the encoder by the axis hardware. There are 2 types of absolute encoder supported by this parameter; SSI and EnDat.

If the number of ENCODER_BITS is to be changed while running, the ATYPE parameter must first be set to zero before entering the new value, then set to the value for SSI.

Value:

Encoder type

Bits

Value

Function

All:

0

0

No data is clocked out of the encoder (default)

SSI:

Bit 0-5

0-32

The number of bits to be clocked out of the encoder.

Bit 6

64

Set for Binary, clear for Gray code (default)

Bit 7

128

Reverses direction (inverts the data bits)

Bit 8-10

256 * n
0<n<7

The number of bits to shift the data.  Use to allow status bits to be ignored.

EnDat:

Bits 0..7

0-255

The total number of data bits returned

Bits 8..13

256 * n
0<n<64

The number of multi-turn bits

Bit 14

16384

This is set by the controller when a correct CRC is calculated from the encoder position data.

Examples:

Example 1:

Set up 2 axes of SSI absolute encoder

ENCODER_BITS AXIS(3) = 12

ENCODER_BITS AXIS(7) = 21

Example 2:

Re-initialise MPOS using absolute value from encoder

SERVO=OFF

ENCODER_BITS = 0

ENCODER_BITS = databits

Example 3:

A 25 bit EnDat encoder has 12 multi-turn and 13 bits/turn resolution. (Total number of bits is 25)

ENCODER_BITS = 25 + (256 * 12)

ATYPE = 47

 

Example 4:

Set up SSI absolute encoder with 21 bits of data and 3 bits of status at bits 21, 22 and 23.  The SSI will clock for 24 bits but only bits 0 to 20 will be transferred to ENCODER and MPOS.

ENCODER_BITS = 21 + (3 * 256)

Example 5:

Renishaw Biss-C linear RL32BAT050B10A encoder with 32 bits of data and 50 nm resolution.  Connected to axis 4 on MC405.  Use Reprogram FPGA … to Select the FPGA that includes Biss-C encoder support.

BASE ( 4 )
' Clear the current axis type
ATYPE = 0
' Clear any previous setting and disables encoder
ENCODER_BITS = 0
WA ( 2 )
' set TO 32 bits, binary data
ENCODER_BITS = 64 + 32
' BiSS Encoder Input (92 = BiSS Servo)
ATYPE = 93
' Short delay to allow encoder to initialise and transmit valid data
WA ( 10 )
' calibrate the axis for mm scale
UNITS = 1 / ( 50 * 10 ^ - 6 )

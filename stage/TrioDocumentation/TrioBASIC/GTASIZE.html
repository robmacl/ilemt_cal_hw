GTASIZE

Type:

System Parameter (Read Only)

Description:

Returns the size of the global target array, which varies across controller types.

The global target array is indexed from zero, so the valid range of indices is 0 to GTASIZE - 1.

This array shares elements between the three target types, TARGET, TARGETJ and TARGETE, so an element may be any one of these types depending on whether they were initialised with GTA, GTAJ, or GTAE.

Value:

The number of elements in the global target array.

Example:

Set the last GTA element to 100, -10, 200, 90, -90, 0.

>>GTA(GTASIZE – 1) = 100, -10, 200, 90, -90, 0

>>

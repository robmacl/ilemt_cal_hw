AXIS_BLENDING

Type:

Axis Parameter

Syntax:

AXIS_BLENDNING AXIS(n) = value

Description:

This parameter controls how advanced speed control moves blend into each other.

Each move may use a different speed, which is set using FORCE_SPEED prior to loading the move.

Additionally, each move may have a different start and end speed, set with STARTMOVE_SPEED and ENDMOVE_SPEED, which by default are disabled and once used may be manually disabled by setting them to -1.

Finally, the rate at which the speed changes from one FORCE_SPEED to another can be set for each move using FORCE_ACCEL and FORCE_DECEL, which may be disabled by setting them to -1, in which case the axis parameters ACCEL and DECEL will be used. For VP_MODEs other than 0, the rate at which acceleration and deceleration are attained can be specified using FORCE_JERK.

Note that the setting of this parameter has an impact on corner speed control functionality, which requires an AXIS_BLENDING of 1 or 3 to function.

Value:

Value

Name

Description

0

Off

Use the move’s FORCE_SPEED only

1

Low (default)

Use the lowest of the previous move’s ENDMOVE_SPEED if set or its FORCE_SPEED (see note), or the next move’s STARTMOVE_SPEED if set or its FORCE_SPEED (see note)

2

Previous

Use the previous move’s ENDMOVE_SPEED if set, otherwise its FORCE_SPEED (see note)

3

Next

Use the next move’s STARTMOVE_SPEED if set, otherwise its FORCE_SPEED (see note)

4

High

Use the highest of the previous move’s ENDMOVE_SPEED if set or its FORCE_SPEED (see note), or the next move’s STARTMOVE_SPEED if set or its FORCE_SPEED (see note)

Note that for VP_MODE = 5, and VP_MODE = 6 with VP_OPTIONS.1 = 1, when a move becomes active, its FORCE_SPEED is loaded into SPEED and SPEED may be assigned new values to adjust the active move’s speed, and AXIS_BLENDING will use the adjusted speed rather than the original FORCE_SPEED.

The terms ‘previous’ and ‘next’ used in the above table is relative to the junction between moves, so ‘previous’ means ahead of the junction and ‘next’ means after the junction.

Examples:

Example 1:

Set the axis to blend using the highest value.

AXIS_BLENDING AXIS(0) = 4

Example 2:

Set the axis to blend using the previous value, instead using BASE rather than the AXIS modifier.

BASE(0)

AXIS_BLENDING = 2

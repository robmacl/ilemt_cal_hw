DIR

Type:

System Command (command line only)

Syntax:

DIR [option]

Alternate Format:

LS [option]

Description:

Prints a list of all programs including their size and RUNTYPE . 

Parameters:

Parameter

Function

none

Directory listing of controller memory

d

Directory listing of SD card memory

u1 or
u2

Directory listing of USB memory stick. (Flex-7 only)

s

Reserved function

x

Extended listing of controller memory (used by Motion Perfect).

V_LIMIT

Type:

Axis Parameter

Syntax:

V_LIMIT AXIS(n) = value

Description:

This parameter sets a speed limit for an axis such that the vector path speed is reduced as required to ensure that the speed of each axis with a limit set does not exceed its limit.

When used with the MOVECIRC and MOVECIRCSP commands, the path speed is constant around the circumference, the rate being determined from the lowest V_LIMIT for each axis in the move.

This parameter is supported by VP_MODE = 0 and VP_MODE = 6, but not by VP_MODE = 5.

Value:

The maximum speed allowed for the specific axis in user UNITS , or a value less than or equal to zero to disable the speed limit.

Examples:

Set up an axis speed limits so that the x-axis operates at no more than 40 mm/s and the y-axis operates at no more than 80 mm/s.

DIM x AS INTEGER = 0

DIM y AS INTEGER = 1

 

V_LIMIT AXIS(x) = 40

V_LIMIT AXIS(y) = 80

 

BASE(x, y)

SPEED = 200

SERVO = ON

 

MOVE(0, 500) 'This moves along the y-axis, so moves at 80 mm/s

WA(100)

MOVE(500, 0) 'This moves along the x-axis, so moves at 40 mm/s

WA(10)

MOVE(500, 500) 'This moves at 45 degrees so VP_SPEED is 56.5685 mm/s

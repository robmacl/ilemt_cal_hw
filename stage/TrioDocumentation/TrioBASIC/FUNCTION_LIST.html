FUNCTION_LIST

Type:

Command and Function

Syntax:

There are two different syntaxes:

First parameter is a string

FUNCTION_LIST(function[, parameter]])

First parameter is not a string

FUNCTION_LIST[([option[, library[, start index[, end index]]]])]

Description:

FUNCTION_LIST outputs a list of currently available FUNCTIONs with parameter and return data definitions.

Parameters:

First parameter is a string

function:

If there is only one parameter, then print the number of parameters for this function.

parameter:

The 0 based index of the function parameter of which to print the data type.

First parameter is not a string

option:

If option is 1 then, print the function names, otherwise print the function details.

library:

If library is empty then, print functions from all libraries, otherwise print functions from the specified library.

start index:

The 0 based index of the first function to print.

end index:

The 0 based index of the last function to print.

 

Examples:

Example 1:

Type FUNCTION_LIST in the Motion Perfect command line terminal:

>>FUNCTION_LIST
f1(param1:INTEGER,param2:INTEGER,param3:BOOLEAN):INTEGER
enable_axes(ax0:INTEGER,ax1:INTEGER,ax2:INTEGER):NONE
disable_axes(ax0:INTEGER,ax1:INTEGER,ax2:INTEGER):NONE
set_rs232(baudrate:INTEGER):NONE
blink_3(op_number:INTEGER):NONE
get_decel_distance(f_decel:FLOAT,f_vel:FLOAT):FLOAT
f_cube(param1:FLOAT):FLOAT
>>

Example 2:

DIM funclist AS STRING(256)

funclist = FUNCTION_LIST

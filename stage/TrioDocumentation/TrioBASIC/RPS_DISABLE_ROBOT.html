DISABLE_ROBOT

Type:

Robotics Command

Syntax:

DISABLE_ROBOT(value)

Description:

This command inhibits any robot programs from running. Any running robot programs will be stopped and move buffers will be cleared.

Parameters:

value: 

TRUE = the command is activated. No robot programs will be running and buffers are cleared.

FALSE = the command is deactivated.

Examples:

Example 1:

When input 12 is triggered, disable robot 1.

ROBOT ( 1 )

WHILE IN ( 12 )= ON
     DISABLE_ROBOT( TRUE )
WEND

ROBOT (- 1 )

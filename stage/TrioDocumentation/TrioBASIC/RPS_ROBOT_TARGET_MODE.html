ROBOT_TARGET_MODE

Type:

Robotics Parameter

Syntax:

ROBOT_TARGET_MODE = value

Description:

ROBOT_TARGET_MODE is used to control level of restrictions to be applied when using GTAE (Global Targets Array - Extended) or TARGETE data type.

ROBOT_TARGET_MODE requires the RPS runtime FEC .

Value:

Bit

Description

Value

1

 

0

Deactivate robot number restriction

2

1

Activate robot number restriction

2

0

Deactivate robot configuration restriction

4

1

Activate robot configuration restriction

3

0

Deactivate object frame restriction

8

1

Activate object frame restriction

4

0

Deactivate tool offset restriction

16

1

Activate tool offset restriction

5

0

Deactivate robot frame restriction

32

1

Activate robot frame restriction

6

0

Deactivate world value of W axis extended range. Default value is between -180 and +180.

64

1

Activate world value of W axis extended range while still respecting joint limits. Value is between -infinity and +infinity (within last joint limits)

Examples:

Example 1:

Deactivate all robot target restrictions.

ROBOT_TARGET_MODE = 0 ' Turn off all restrictions for TARGETEs

Example 2:

Monitor robot used in the target.

ROBOT_TARGET_MODE.1 = 1   ' Turn on Robot Number Restriction.  

Example 3:

Monitor robot used in the target and the robot configuration for that target.

ROBOT_TARGET_MODE = 6   ' Turn on robot number and robot conf restriction.

OPEN_WIN

Type:

Axis Parameter

Alternate Format:

OW

Description:

This parameter defines the start position of the first registration window which will be used for registration marks if windowing is specified by the REGIST command.

Value:

Absolute start position of the first registration window in user units.

Example:

Enable registration but only look for registration marks between 170 mm and 230 mm

OPEN_WIN = 170.00

CLOSE_WIN = 230.0

REGIST(256 + 3)

WAIT UNTIL MARK

CPU_TEMP_MAX

Type:

System Parameter

Syntax:

value = CPU_TEMP_MAX

CPU_TEMP_MAX = value

Description:

This keeps a record of the highest temperature the CPU has reached.

It can be reset by writing a value to it in degrees Celsius (°C), usually zero.

CPU_TEMP_MAX is not available for Euro404/408, MC403, MC403-Z, MC405, MC405-Y, MC4NE and MC4N-RTEX.

Examples

Example 1:

Display the maximum temperature the CPU has reached on the command line.

PRINT CPU_TEMP_MAX

Will display:

58.1

Example 2:

Reset the maximum temperature.

CPU_TEMP_MAX = 0

CLEAR_BIT

Type:

Logical and Bitwise operator

Syntax:

CLEAR_BIT(bit, vr_index [, mode] )

Description:

CLEAR_BIT can be used to clear the value of a single bit within a VR variable.

You can change the mode for clear ing a single bit to TABLE .

Parameters:

bit:

The bit number to clear, valid range is 0 to 52

variable:

The VR on which to operate

Mode:

0

Bit resets to VR (default)

1

Bit resets to TABLE

Examples:

Example 1:

Set bit 6 in VR (23) to zero.

CLEAR_BIT(6, 23)

Example 2:

Set bit 6 in TABLE (23) to zero.

CLEAR_BIT(6, 23, 1)

UNIT_SW_VERSION

Type:

Slot Parameter

Syntax:

value = UNIT_SW_VERSION SLOT(n)

Description:

Returns a value that is the version number of the Software in a module fitted at the specified SLOT.  This parameter can be used when connected to a MC664(X) to get the Software version of each module fitted to the Motion Coordinator.

For FPGA information use FPGA_VERSION

Example:

>>PRINT UNIT_SW_VERSION SLOT(0)

0

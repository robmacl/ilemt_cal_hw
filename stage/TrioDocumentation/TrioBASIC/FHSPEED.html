FHSPEED

Type:

Axis Parameter

Description:

The feed-hold speed defines the speed to use when the feed-hold input ( FHOLD_IN ) is active.

Value:

The speed in user units (default 0).

Example:

Specify that motion is to ramp down to a tenth of the axis speed when the feed-hold input is active.

BASE(3)

SPEED = 1000

FHSPEED = 0.1 * SPEED

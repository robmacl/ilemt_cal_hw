PROCESS

Type:

System Command (Command line only)

Description:

Displays information about the running processes.

There are some system processes that you cannot stop.

Returned Values:

Process:

The process number

Type:

The Type of process executing

Status:

The execution state of the process

Program:

The name of the program running in the process

Line:

The line number of a program that is executing

Time:

The length of time that the process has been running

CPU:

The percentage of CPU time used by the process

From V2.0298 onwards, when an IEC program is running, the multi-tasking is flat with no fast processes.  Use SCHEDULE_TYPE.8 = 1 in MC_CONFIG to enable fast processes with IEC 61131-3.

Example:

Check the state of the processes in the command line.

>>process

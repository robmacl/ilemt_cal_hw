STARTMOVE_SPEED

Type:

Axis Parameter

Description:

This parameter sets the start speed for a motion command that supports advanced speed control (commands ending in SP). Subject to the AXIS_BLENDING mode, the VP_SPEED will decelerate so that STARTMOVE_SPEED is reached at the start of the motion command.

STARTMOVE_SPEED is loaded into the buffer at the same time as the move so you can set different speeds for subsequent moves.

STARTMOVE_SPEED has a default value of -1 to disable its use.

In general, STARTMOVE_SPEED is only used by the CORNER_MODE methods. The user can program all profiles using only FORCE_SPEED and ENDMOVE_SPEED .

Value:

The speed at which the SP motion command will start, in user UNITS (default -1).

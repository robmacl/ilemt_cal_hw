LENB

Type:

String Function

Syntax:

value = LENB(string)

Description:

Returns number of bytes used in the specified string.

Note that this will differ from the number of characters in the string when UTF-8 characters are present.

Parameters:

string:

String to be measured.

Examples:

Example 1:

Define a variable of type string, assign to it a string containing UTF-8 characters, and then display its byte length and character length.

DIM str1 AS STRING(30)

str1 = "Здравей свят" '"Hello world" in Bulgarian

PRINT LENB(str1), LEN(str1) 'Will display 23 and 12

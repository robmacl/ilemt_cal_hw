ASIN

Type:

Mathematical Function

Syntax:

value = ASIN(expression)

Alternate Format:

ASN(expression)

Description:

Returns the arc-sine (also known as inverse sine) of a number which should be in the range -1 to +1, the result being an angle measured in radians in the range -π/2 to π/2 (see PI ).

Parameters:

value:

The angle measured in radians

expression:

Any valid TrioBASIC numeric expression with a value in the range -1 and +1

Example:

Print the arc-sine of -1 on the command line

>>PRINT ASIN(-1)

-1.5708

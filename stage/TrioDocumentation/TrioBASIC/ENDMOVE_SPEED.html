ENDMOVE_SPEED

Type:

Axis Parameter

Description:

This parameter sets the end speed for a motion command that supports advanced speed control (commands ending in SP). Subject to the AXIS_BLENDING mode, the VP_SPEED will decelerate so that ENDMOVE_SPEED is reached at the end of the motion command.

ENDMOVE_SPEED is loaded into the buffer at the same time as the move so you can set different speeds for subsequent moves. If there are no further motion commands in the buffer the current move will decelerate to a stop.

ENDMOVE_SPEED has a default value of -1 to disable its use.

Value:

The speed at which the SP motion command will end, in user UNITS (default -1).

Examples:

Example 1:

In this example the controller will start ramping down the speed (at the specified rate of DECEL ) so at the end of the MOVESP (20) the VP_SPEED =10. The next move continues with a FORCE_SPEED of 10. The final ENDMOVE_SPEED is overwritten to zero as there are no more buffered moves.

FORCE_SPEED = 15

ENDMOVE_SPEED = 10

MOVESP(20)

FORCE_SPEED = 10

ENDMOVE_SPEED = 5

MOVESP(5)

Example 2:

A machine can merge interpolated moves however it must slow down to 50% of the speed for the transition.

FORCE_SPEED = 1000

ENDMOVE_SPEED = 500 '50% of FORCE_SPEED

MOVESP(100, 10)

MOVESP(70, -10)

MOVESP(120, 15)

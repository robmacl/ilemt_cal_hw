MOD

Type:

Mathematical Operator

Syntax:

value = expression1 MOD expression2

Description:

Returns the integer modulus of an expression, this is the value after the integer has wrapped around the modulus

Parameters:

value:

the modulus of expression 1

expression1:

Any valid TrioBASIC expression used as the value to apply the modulus to.

expression2:

Any valid TrioBASIC expression used as the modulus

Example:

Use the MOD 12 to turn a 24 hour value into 12 hour.

>>PRINT 18 MOD 12

6.0000

>>

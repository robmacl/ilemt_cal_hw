KINEMATIC_GROUP_IO

Type:

Robotics Command

Syntax:

KINEMATIC_GROUP_IO(group, KMI, KAI, ESI, DMI, REI, SAI, DEI, MEI, MPO, REO, ERO)

 

KINEMATIC_GROUP_IO(group)

Description:

This command allows the setting of the digital I/O used by the RPS 2.0 kernel.

KINEMATIC_GROUP_IO requires the RPS runtime FEC

RPS 2.0 kernel must be activated using the RPS_KERNEL_V2 keyword to enable its use.

Parameters:

group:           

The group number, 0-7.

When used as the only KINEMATIC_GROUP_IO parameter, it prints the existing settings to the currently selected output channel (default channel 0).

KMI:

Key switch manual input

KAI:

Key switch automatic input

ESI:

Emergency stop input

DMI:

Demand input

REI:

Reset errors input

SAI:

Safety input

DEI:

Drive enabled input

MEI:

Master enable input

MPO:

Motor powered output

REO:

Robot energised output

ERO:

Error output

Examples:

Example 1:

Set relevant I/O for Kinematic Group 0.

>>kinematic_group_io (0, 6, 7, 20, 8, 9, 21, 10, 11, 12, 1, 2) 

Example 2:

Print I/O points being used for Kinematic Group 0

>>kinematic_group_io(0) 
0 [ 6, 7, 20, 8, 9, 21, 10, 11, 12, 1, 2]

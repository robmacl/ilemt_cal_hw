IN_POS_SPEED

Type:

Axis Parameter

Syntax:

IN_POS_SPEED AXIS(n) = <value>

Description:

This axis parameter is a speed, scaled using axis UNITS, indicating the required minimum measured speed before IN_POS signals that the axis move has finished and that the axis is in position.

The IN_POS_SPEED defaults to zero. Reaching zero position and speed error may take considerable time or not happen if this parameter is not set.

Value:

The speed value in the units set for the axis

Example:

Make a move and wait for the axis to be in position before continuing.

BASE(0)
SERVO=ON
UNITS=100000
SPEED=5
ACCEL=50
DECEL=50

IN_POS_DIST=0.002
IN_POS_SPEED=0.1
REPEAT
TRIGGER
  MOVE(1)
  WAIT IN_POS
  WA(50)
UNTIL FALSE

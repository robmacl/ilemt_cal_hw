SGN

Type:

Mathematical Function

Syntax:

value = SGN(expression)

Description:

The SGN function returns the sign of a number.

Parameters:

value:

 

 

1

Positive non-zero

0

Zero

-1

Negative

expression:

Any valid TrioBASIC expression.

Example:

Detect the sign of the number -1.2 using the command line.

>>PRINT SGN(-1.2)

-1.0000

>>

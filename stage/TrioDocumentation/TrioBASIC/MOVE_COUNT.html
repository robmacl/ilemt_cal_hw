MOVE_COUNT

Type:

Axis Parameter

Description:

MOVE_COUNT increments every time a motion command loads into the MTYPE buffer or when a command is automatically re-loaded such as MOVELINK , CAMBOX and FLEXLINK when using option value 4.  (Auto Repeat)

MOVE_COUNT can be written to set an initial value.

Value:

The number of movements loaded into the MTYPE buffer.

A move with zero length will not be loaded into the MTYPE buffer and therefore will not be counted.

Fwd and Rev Jog moves are not counted.

Examples:

Example1:

Run the motion program and then turn on the OP(11) after 10 moves have been loaded.

MOVE_COUNT = 0

RUN "MOTION"

WAIT UNTIL MOVE_COUNT > 10

OP(11,ON)

Example2:

Run a continuous MOVELINK and stop it after 250 cycles, stopping at the end of the last complete MOVELINK length.

motion_axis = 0

link_axis = 1

BASE(motion_axis)

MOVE_COUNT = 0

MOVELINK(10,12,2,2,link_axis,4)

IF IDLE AXIS(link_axis) THEN

    FORWARD AXIS(link_axis)

ENDIF

WAIT UNTIL MOVE_COUNT >= 250

REP_OPTION.1 = 1

WAIT IDLE

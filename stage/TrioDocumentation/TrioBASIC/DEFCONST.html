DEFCONST

Type:

Program Constant

Syntax:

DEFCONST name value

Description:

DEFCONST defines a literal name with a value, either integer or floating point.

These literals can then be used with array declarations or switch statements or any general line of BASIC.  They can be added to INCLUDE files and referenced within the INCLUDE file itself or the main program that uses the INCLUDE file.

DEFCONST is evaluated at compile time. This limits its use for certain runtime functionality. For example, “DEFCONST process_number PROCNUMBER” will not work since DEFCONST does not know which process the program is on in runtime.

Parameters:

name:

Any user-defined name containing lower case alpha, numerical or underscore (_) characters.

value:

The value or evaluated expression assigned to the name.

Examples:

Example 1:

Declare 4 constant literals.

DEFCONST mysize 20

DEFCONST fval SIN(90)

DEFCONST i1 7

DEFCONST i2 i1 * 1000

 

DIM myarray AS INTEGER(mysize)

Example 2:

Use a constant with SELECT_CASE.

DEFCONST errorval 100

 

SELECT_CASE swval

 

    CASE 0

        PRINT "valid data"

 

    CASE errorval

        PRINT "SWVAL invalid"

END_CASE

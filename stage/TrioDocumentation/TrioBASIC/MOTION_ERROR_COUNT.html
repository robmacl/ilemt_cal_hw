MOTION_ERROR_COUNT

Type:

System Parameter (read only)

Syntax:

value = MOTION_ERROR_COUNT

Description:

This system parameter returns an integer indicating the number of motion errors present.

Value:

The number of bits set in MOTION_ERROR.

Example:

Display the number of motion errors on the command line:

>>?MOTION_ERROR_COUNT
0

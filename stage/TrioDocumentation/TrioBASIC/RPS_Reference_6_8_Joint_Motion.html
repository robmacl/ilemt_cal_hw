Joint Motion

The motion commands MOVEJ and MOVEJREL are used to move the robot from one point to another along a non-linear path. All axes reach the destination position at the same time. It is the fastest type of movement due to the axes move the exact number of degrees needed to reach the desired position.

The speed of the move is an interpolated speed and NOT the speed of any individual axis or the end effector.

For example, a robot is to move from 10.5,0,20,0,0,0 in joint position to 90.21,44.2,24.1,4.1,0.0,12.2 with the speed 50.

The total position change is 144.33 as detailed below:

J1: 90.21 - 10.5 = 79.71

J2: 44.22 – 0 = 44.22

J3: 24.1 – 20 = 4.1

J4: 4.1 - 0 = 4.1

J5: 0 – 0 = 0

J6: 12.2 – 0 = 12.2

The robot will cover the distance 144.33 degrees with the speed 50. Individual axes may travel at a higher or a lower speed than 50 but all of them will reach end position at the same time.

MOVEJ

This command uses absolute coordinates.

Syntax

MOVEJ target [S:= T:= O:= C:= Z:= A:= D:=]

Examples

Example 1

GTA(0) = 350, 0, 50, 180, 0, 0, "p1"
GTA(1) = 400, 100, 20, 180, 0, 0, "p2"
GTA(2) = 450, -300, 60, 180, 0, 0, "p3"
OBJECT_FRAME(1,"of1", -10, 0, 0, 0, 0, 15)
TOOL_OFFSET(1,"tool1", 0, 0, 50, 0, 0, 0)

ROBOT(0)
MOVEJ GTA("p1")
MOVEJ GTA("p2")
MOVEJ GTA("p3")
STOP

The three move instructions commands Robot 0 to move linearly to target point 2 (GTA 2 with name “p3”) from target point 0 (GTA 0 with name “p1”). Note that there are no other embedded parameters in any of the MOVEJ instructions. This instruction will therefore move the robot using the following parameters.

The object frame and tool offset declarations above do not automatically set the object frame and tool offset as the currently active object frame and tool offset. The currently active object frame and tool offset have not been modified.

MOVEJREL

This command uses relative coordinates.

A single joint can be moved by simply set all values of the target as 0 except the desired joint to move.

Syntax

MOVEJREL target [S:= T:= O:= C:= Z:= A:= D:=]

Examples

Example 1

GTA(0) = 0,-20,0,0,0,0

ROBOT(0)
MOVEJREL GTA(0) 'The robot will move 20 degrees of the second joint in negative direction from its current position
STOP

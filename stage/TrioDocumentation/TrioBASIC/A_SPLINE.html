A_SPLINE

Type:

Command

Syntax:

A_SPLINE(parameters)

Description:

Generate the Akima spline interpolation of the given set of points.

Parameters:

data_in

Index to the TABLE data that contains the points to be interpolated

points

Number of points to be interpolated

data_out

Index to the TABLE data that will contain the interpolated points

expansion

Number of output points between each input point

Example:

TABLE ( 0 , 100 , 138.1773291 , 49.315059 , - 84.8872489 , - 141.0446116 , - 67.5262089 )
A_SPLINE ( 0 , 6 , 100 , 10 )
CAMBOX ( 100 , 159 , 1 , 6 , 1 )

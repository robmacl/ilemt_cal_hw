RND

Type:

Mathematical Function

Syntax:

value = RND

value = RND(<upper-limit>)

value = RND(<lower-limit>, <upper-limit>)

Description:

The RND function returns a pseudo-random unsigned integer value, either using the full range of a 32-bit unsigned integer with no parameters, or a value between 0 and limit-1 if an upper limit is specified.

Note that the sequence generated varies with each program execution, which is usually desirable, but if you require the same sequence generated then use the RANDOMIZE command.

Parameters:

lower-limit:

Optional parameter to specify the lower inclusive limit of the random value, which if omitted defaults to 0 (zero)

upper-limit

Optional parameter to specify the exclusive upper limit of the random value

value:

A pseudo-random unsigned integer value

Examples:

Example 1:

Print a random 8-bit number on the command line

>>PRINT RND(1<<8)

173

>>PRINT RND(1<<8)

98

>>PRINT RND(1<<8)

225

>> 

Example 2:

Print a random number from 0 to 99 inclusive on the command line

>>PRINT RND(100)

61

>>PRINT RND(100)

3

>>PRINT RND(100)

40

>> 

Example 3:

Print a random number from -100 to 99 inclusive on the command line

>>PRINT RND(-100, 100)

76

>>PRINT RND(-100, 100)

-17

>>PRINT RND(-100, 100)

-2

>>

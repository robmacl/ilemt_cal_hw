NTYPE

Type:

Axis Parameter (Read Only)

Description:

This parameter holds the type of the first buffered move.

The NTYPE buffer can be cleared using CANCEL (1)

Value:

The numerical value of the move type

See MTYPE for a list of return values.

Example:

Example 1:

If the first move buffer (NTYPE) is empty apply another move from a table

IF NTYPE = 0 THEN

  MOVE( TABLE(count))

  count = count + 1

ENDIF

In the MC4/5/6 ranges, the preferred axis parameter is LOADED .  The example below will work more reliably than by using NTYPE.

IF LOADED THEN

  MOVE( TABLE(count))

  count = count + 1

ENDIF

Example 2:

In the command line (Terminal #0) check the status of MTYPE, NTYPE and PMOVE of process 14.

>>?NTYPE

2

>>?MTYPE

1

>>?PMOVE PROC(14)
0

>>

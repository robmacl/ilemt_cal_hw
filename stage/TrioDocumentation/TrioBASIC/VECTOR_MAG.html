VECTOR_MAG

Type:

Mathematical Function

Syntax:

value = VECTOR_MAG(expr-1[, expr-2[, ..., expr-6]])

Alternate Format:

value = HYPOT(expr-1[, expr-2[, ..., expr-6]])

Description:

The VECTOR_MAG function returns the magnitude of a vector defined by its components, which can number from 1 to 6.

The vector’s magnitude is the square root of the sum of the squares of its components, so it is the same as the hypotenuse of a 2-D or 3-D triangle, hence its alternate function name of "HYPOT".

Parameters:

expr-n:

The floating-point value of component ‘n’ of the vector.

Examples:

Example 1:

Print the magnitude of a scalar value:

>>PRINT VECTOR_MAG(PI)

3.1416

Example 2:

Print the magnitude of a two-dimensional vector:

>>PRINT VECTOR_MAG(3, 4)

5.0000

Example 3:

Print the magnitude of a three-dimensional vector:

>>PRINT VECTOR_MAG(3, 4, 12)

13.0000

Example 4:

Print the magnitude of a six-dimensional vector:

>>PRINT VECTOR_MAG(1, 2, 3, 4, 5, 6)

9.5394

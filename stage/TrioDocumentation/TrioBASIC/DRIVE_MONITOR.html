DRIVE_MONITOR

Type:

Axis Parameter

Syntax:

value = DRIVE_MONITOR AXIS(n)

Description:

Returns current value of parameter identified using DRIVE_CONTROL.

Example:

To read the actual following error in the drive, one would do the following:

DRIVE_CONTROL = 5

fe_actual = DRIVE_MONITOR

 

See Also:

DRIVE_READ , DRIVE_WRITE , DRIVE_CONTROL

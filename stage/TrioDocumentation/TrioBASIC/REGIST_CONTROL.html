REGIST_CONTROL

Type:

Axis Parameter

Syntax:

REGIST_CONTROL = value

Variable = REGIST_CONTROL

Description:

Read or set the low-level bit pattern in the registration control register. REGIST_CONTROL is an 8-bit axis parameter that conforms to the standard use of BASE and the AXIS modifier.

Parameters:

value:

Bit

Description

Chan A

0

Input (1) or Z-mark (0)

1

Not used

2

Enable

3

Falling (1) or rising (0)

Chan B

4

Input (1) or Z-mark (0)

5

Not used

6

Enable

7

Falling (1) or rising (0)

 

Example 1:

Set up registration on falling edge channel B using REGIST function 20. Then check that the registration mode has been set correctly.

BASE(2)

REGIST(20, 1, 0, 1, 0)

IF REGIST_CONTROL.4 = 1 THEN PRINT "Input selected"

IF REGIST_CONTROL.6 = 1 THEN PRINT "Registration enabled"

IF REGIST_CONTROL.7 = 1 THEN PRINT "Falling edge selected"

 

Example 2:

Disable a registration channel so that any signal received will NOT cause the mark to be TRUE and will NOT store a REG_POS.

BASE(0)

REGIST_CONTROL.2 = 0 'Disable channel A regist but leave channel B active

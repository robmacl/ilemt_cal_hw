MPE

Type:

System Command

Syntax:

MPE(mode)

Description:

Sets the type of channel handshaking to be performed on the command line.

This is normally only used by the Motion Perfect program, but can be used for user applications with the PCMotion ActiveX control in asynchronous mode.

There is only one Command Line process.  If Trio PCMotion connects and uses the Motion Perfect command line port, then Motion Perfect will not be able to connect at the same time.

Parameters:

mode:

0

No protocol.  Motion Perfect is in Direct Mode or not connected.

1

No longer used.

2

No longer used.

3

Motion Perfect version 3 connected in either tool or synchronised mode.

4

Motion Perfect version 4 connected in tool mode.

5

Motion Perfect version 4 connected in synchronised mode.

Example:

Check the mode before attempting to reset the Motion Coordinator.

IF MPE = 0 THEN

  ' OK to reset the controller

  EX

ELSE

  PRINT #5,"Cannot reset when Motion Perfect is connected"

  ENDIF

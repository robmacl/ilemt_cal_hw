ROBOT_INPUT_INDEX

Type:

Robotics Parameter (Read Only)

Syntax:

ROBOT_INPUT_INDEX(group, index)

Description:

Returns the digital input channel for the given frame group’s input.

ROBOT_IO requires the RPS runtime FEC

RPS 2.0 kernel must be activated using the RPS_KERNEL_V2 keyword to enable its use.

Parameters:

group: 

The group number, 0-7.

index:

Robot input index as shown in table below:

Index

Code

Meaning

0

KMI

Key switch manual input

1

KAI

Key switch automatic input

2

ESI

Emergency stop input

3

DMI

Demand input

4

REI

Reset errors input

5

SAI

Safety input

6

DEI

Drive enabled input

7

MEI

Master enable input

 

 

Examples:

Example 1:

Print the input channel mapped to drive enabled input (DEI).

Terminal

>>?ROBOT_INPUT_INDEX(0,6)

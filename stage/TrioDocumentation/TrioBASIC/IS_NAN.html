IS_NAN

Type:

Mathematical Function

Syntax:

value = IS_NAN(expression)

Description:

Check an expression to test if it is a NaN (not a number, a bit pattern that does not represent a valid floating-point value).

If a battery backed memory loses its battery power, the bit pattern will be random and the NaN will cause a BASIC error.

NaN values can also be set from an external source like a PC or PLC setting values over a fieldbus.

Parameters:

expression

Any valid numeric expression

Examples:

Example 1:

Check all the VRs in a MC464 and set any that are NaN to 0.

FOR i = 0 TO 65535

    IF IS_NAN(VR(i)) THEN

        VR(i) = 0

    ENDIF

NEXT i

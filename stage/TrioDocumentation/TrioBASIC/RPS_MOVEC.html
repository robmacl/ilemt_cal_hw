MOVEC

Type:

Robotics Command

Syntax:

MOVEC centre_point end_point [S:= T:= O:=]

Description:

MOVEC is used to move the robot from one point to another along a circular path. All axes reach the destination position at the same time.

KINEMATIC_GROUP requires the RPS runtime FEC

Parameters:

Parameter

Data type

Description

centre_point

TARGET

Cartesian target that defines the centre of the circular move. This point is traversed along the executed circular move.

NOTE: joint target is not suitable for this move. Error ‘Invalid mix of data type’ will be generated if joint target is used instead.

end_point

TARGET

Cartesian target that defines the end of the circular move.

NOTE: joint target is not suitable for this move. Error ‘Invalid mix of data type’ will be generated if joint target is used instead.

Embedded parameters:

 

See RPS Move Commands for details

Examples:

Example 1:

GTA(0) = 350, 0, 50, 180, 0, 0, “p1”
GTA(1) = 400, 100, 20, 180, 0, 0, “p2”
GTA(2) = 450, 0, 60, 180, 0, 0, “p3”
OBJECT_FRAME(1,"of1", -10, 0, 0, 0, 0, 15)
TOOL_OFFSET(1,"tool1", 0, 0, 50, 0, 0, 0)

ROBOT(0)

MOVEL GTA("p1")
MOVEC GTA("p2") GTA("p3")
STOP

This example commands Robot 0 to Move linearly to p1, followed by circular move towards p3 along p2. Note that there are no other embedded instructions in the code. Thus, the robot will execute the code using the following parameters.

Example 8:

Robot needs to make a print of letter ‘R’ using MOVEC and MOVEL. To make perfect half circle, the radius of the circle needs to be half the distance between the starting point and the end point.

ROBOT(0)
WDOG = 1

DIM r1, r2, r3, r4, r5, r6, r7, r8, r9, r10 AS TARGET

r1 = 150, -150, 120, 0, 0, 0
r2 = 150, -110, 120, 0, 0, 0
r3 = 170, -90, 120, 0, 0, 0
r4 = 190, -110, 120, 0, 0, 0
r5 = 230, -90, 120, 0, 0, 0
r6 = 230, -110, 120, 0, 0, 0
r7 = 190, -130, 120, 0, 0, 0
r8 = 230, -130, 120, 0, 0, 0
r9 = 230, -150, 120, 0, 0, 0
r10 = 150, -150, 120, 0, 0, 0

OBJECT_FRAME(1, "wobj1", 10, 10, 0, 0, 0, 0)
TOOL_OFFSET(1, "gripper", 10, 10, 0, 0, 0, 0)

MOVEJ r1 S:=15 O:=1 T:=1 A:=VR(0) D:=VR(1)
WAIT IDLE AXIS(0)

MOVEL r2 S:=50 O:=1 T:=1 A:=VR(0) D:=VR(1)
WAIT IDLE AXIS(0)

MOVEC r3 r4 S:=50 O:=1 T:=1 A:=VR(0) D:=VR(1)
WAIT IDLE AXIS(0)

MOVEL r5 S:=50 O:=1 T:=1 A:=VR(0) D:=VR(1)
WAIT IDLE AXIS(0)

MOVEL r6 S:=50 O:=1 T:=1 A:=VR(0) D:=VR(1)
WAIT IDLE AXIS(0)

MOVEL r7 S:=50 O:=1 T:=1 A:=VR(0) D:=VR(1)
WAIT IDLE AXIS(0)

MOVEL r8 S:=50 O:=1 T:=1 A:=VR(0) D:=VR(1)
WAIT IDLE AXIS(0)

MOVEL r9 S:=50 O:=1 T:=1 A:=VR(0) D:=VR(1)
WAIT IDLE AXIS(0)

MOVEL r10 S:=50 O:=1 T:=1 A:=VR(0) D:=VR(1)
WAIT IDLE AXIS(0)

WDOG = 0

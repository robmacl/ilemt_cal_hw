ACCEL_FACTOR

Type:

Axis Parameter

Description:

This parameter is used to reduce the acceleration and deceleration specified in the ACCEL and DECEL axis parameters by the per-unit value entered when VP_MODE is 5 or 6.

Acceleration factor range is 0-1, where 1 represents using 100% of the acceleration value.

Examples:

Example 1

An analogue input from a potentiometer is used to adjust the acceleration of the motion on axes 2.

IF VR(200) = 1 THEN

  'Start the acceleration control routine

  BASE(2)

  REPEAT

    ACCEL_FACTOR = AIN(0) / 4096   

    WA(100)

  UNTIL VR(200) = 0

ENDIF

 

Example 2

Set acceleration of axis 4 to 50% of the programmed acceleration.

ACCEL_FACTOR AXIS(4) = 0.5

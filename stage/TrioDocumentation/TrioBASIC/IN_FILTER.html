IN_FILTER

Type:

I/O command

Syntax:

value = IN_FILTER(input_no[, final_input], filter_value)

Description:

IN_FILTER is used configure digital filtering on digital Inputs. Digital I/O refers to the 24 V inputs and outputs attached to the Motion Coordinator. Some controllers allow for a filter to be configured on these inputs.

Filtering is only supported on MotionPLC controllers and MotionPLC Slices currently.

Parameters:

input_no:

Input to return the value of/start of input group

final_input:

Last input of group

filter_value:

Filtering range. The input state (ON/OFF) is registered only after it has remained stable for the duration of the filter time.

Description

Value

No filtering

0

125 µs

1

250 µs

2

500 µs

1 ms

4

2 ms

4 ms

6

7

8

Examples:

Example 1:

Set up input filtering to filter at 2 ms on input number 0:

IN_FILTER(0, 5)

Example 2:

Set up input filtering to filter inputs at 8 ms for a range of inputs

filter_8ms = 7
IN_FILTER(0, 4, filter_8ms)

Example 3:

Turn off all filtering on inputs 0 to 7.

IN_FILTER(0, 7, 0)

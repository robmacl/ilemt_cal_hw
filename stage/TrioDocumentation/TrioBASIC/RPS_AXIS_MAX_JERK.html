AXIS_MAX_JERK

Type:

Robotics Parameter

Description:

The AXIS_MAX_JERK axis parameter can be used to read and write the maximum possible axis jerk.

JERK is the derivative of acceleration. JERK units are /sec/sec/sec.  The way JERK is applied to the axis motion depends on the value of VP_MODE .

Value:

The robot axis maximum JERK in AXIS_UNITS / sec 3.

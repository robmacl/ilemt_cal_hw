EXIT_LOOP

Type:

Program Structure

Syntax:

EXIT_LOOP

Description:

This command applies to the loop structures WHILE…WEND, FOR TO…NEXT and REPEAT…UNTIL.

Once this command is executed within one of the structures mentioned above, the execution will jump out of the loop, therefore, the following iterations will never be performed.

Example:

While input 12 is off, move the base axis and flash an LED on output 10, if input 9 is active, regardless of the state of input 12, the loop will terminate after the base axis moves 200.

WHILE IN(12) = OFF

  MOVE(200)

  WAIT IDLE

  IF IN(9) = ON THEN

    EXIT_LOOP

  ENDIF 

  OP(10, OFF)

  MOVE(-200)

  WAIT IDLE

  OP(10, ON)

WEND

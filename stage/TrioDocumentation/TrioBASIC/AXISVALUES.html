AXISVALUES

Type:

Axis Command

Syntax:

AXISVALUES(axis, bank)

Description:

Used by Motion Perfect to read a bank of axis parameters.

The data is returned in the format:

<Parameter> <type>=<value>

<Parameter> is the name of the parameter

<type> is the type of the value:         

i

Integer.

F

Float.

S

String.

C

String of letters, where upper case letters indicate an error.

<value> is an integer, float or string depending on the type

Parameters:

axis:

The axis number where you want to read the parameters

bank:

The bank of parameters that you wish to read.

0

Displays the data that is only adjusted through TrioBASIC.

1

Displays the data that is changed by the motion generator.

<  Less Than 

 Type:

Comparison Operator

Syntax:

<expression-1> < <expression-2>

Description:

Returns TRUE if <expression-1> is less than <expression-2>, otherwise returns FALSE .

The comparison operators are typically used in conditional statements like IF , REPEAT and WHILE , though can be used anywhere an expression is valid.

Parameters:

Expression1:

Any valid TrioBASIC expression

Expression2:

Any valid TrioBASIC expression

Example:

Check that the value from analogue input 1 is less than 10, and if so, then execute the subroutine ‘rollup’.

IF AIN(1) < 10 THEN GOSUB rollup

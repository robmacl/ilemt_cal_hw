DRIVE_BRAKE_OUTPUT

Type:

Axis Parameter

Alternate Format:

None

Description:

Assign the Motion Coordinator output to operate as the brake control.  The digital output allocated to this function will not be available for use in programs via OP command, PSWITCH or as a PLC output.

Brake control sequences and timing are set in the drive using configuration software.  For example in the Motion Perfect DX5 setup pages.

DRIVE_BRAKE_OUTPUT supports specific EtherCAT “slim” drives where brake control is delegated to the EtherCAT master.

Value:

Output number within the range 0 to 1023.

Example:

Set up a 3 axis system to control the motor brakes using outputs 32 to 34 on a Flexslice Output module.

DRIVE_BRAKE_OUTPUT AXIS(0) = 32
DRIVE_BRAKE_OUTPUT AXIS(1) = 33
DRIVE_BRAKE_OUTPUT AXIS(2) = 34

TrioBASIC Errors

TrioBASIC programs can generate errors both as a consequence of errors in the program structure or type of variables used or as the result of a command being unsuccessful.  Most errors are reported using a channel 9 status message which will be displayed as an error in Motion Perfect. Some commands return a value which indicates the error status. The table below gives the possible values for all error codes generated by the controller.

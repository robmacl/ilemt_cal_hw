JERK

Type:

Axis Parameter

Syntax:

JERK AXIS(n) = <value>

Description:

The JERK axis parameter is the derivative of acceleration. JERK units are /sec/sec/sec and specifies the rate at which acceleration and deceleration should change. JERK is applied only when using VP_MODE 5 or 6, with 0 using SRAMP to smooth motion.

As of firmware version 2.0342, the SRAMP keyword may be used instead of JERK, but if SRAMP is assigned a value (even 0), then any assignment to JERK will be ignored. When SRAMP is used, the JERK is calculated from ACCEL / SRAMP.

Using SRAMP with VP_MODE = 6 is intended solely to aid portability for existing applications, and it is strongly recommended that new applications use JERK.

Example:

VP_MODE = 6
UNITS = 1000
SPEED = 50000
ACCEL = 80000
DECEL = 80000
JERK = 300000

MOVE(50000)

IEEE_IN

Type:

Mathematical Function

Syntax:

IEEE_IN(byte0, byte1, byte2, byte3[, byte4, byte5, byte6, byte7])

Description:

The IEEE_IN function returns the floating-point number represented by the given byte values in IEEE 754 format, typically received over a communications link such as Modbus. An IEEE 754 single-precision value is composed of 4 bytes, and a double-precision value is composed of 8 bytes.

Parameters:

byte0 - 7:

The bytes that represent an IEEE 754 floating point value, 4 values for 32-bit float and 8 values for a 64-bit float.

Byte 0 is the high byte of the IEEE 754 floating-point format.

Examples:

Example 1:

Take 4 bytes that have been sent over Modbus to VR s and recombine them into a floating-point number.

VR(200) = IEEE_IN(VR(0), VR(1), VR(2), VR(3))

Example 2:

Take 8 bytes that have been sent over Modbus to VR s and recombine them into a floating-point number.

VR(200) = IEEE_IN(VR(0), (1), VR(2), VR(3), _

                  VR(4), VR(5), VR(6), VR(7))

BREAK_DELETE

Type:

System Command (command line only)

Syntax:

BREAK_DELETE "program name" line_number

Description:

Used by Motion Perfect to remove a break point from the specified program at the specified line number.

If a non-existent line number is selected (e.g., line 50 when the program only has 40 lines), the controller will return an error.

Parameters:

program name:

the name of any program existing on your controller

line_number:

the line umber where to remove the breakpoint

Example:

Remove the break point at line 8 of program "simpletest"

BREAK_DELETE "simpletest" 8

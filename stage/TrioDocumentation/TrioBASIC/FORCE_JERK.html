FORCE_JERK

Type:

Axis Parameter

Description:

This parameter sets the JERK for a motion command that supports the advanced speed control. The VP_SPEED will have a rate of change of acceleration/deceleration that is controlled by the FORCE_JERK value.

With VP_MODE = 5 and VP_MODE = 6 with VP_OPTIONS.1 = 1, the forced demands (those loaded with the move) are bound to the axis demands such that the axis demands are primed with the forced demands of a move when it becomes active. This binding allows the axis demands to be adjusted, and the changes made influence the current move’s profiling.

FORCE_JERK is also used to provide the jerk demand to MOVEADD.

Value:

The jerk value at which the SP motion command will change acceleration/deceleration by. The default value is -1 and means the parameter is not used.

Examples:

Example 1:

With continuous motion running on the axis, make an additional move to advance the axis by 40 mm after setting the speed, acceleration, deceleration, and jerk values for the added movement.

FORWARD

WA(100)

 

FORCE_SPEED = 50

FORCE_ACCEL = 2000

FORCE_DECEL = 2000

FORCE_JERK = 50000

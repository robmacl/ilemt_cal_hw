+  Add

Type:

Mathematical operator

Syntax:

<expression1> + <expression2>

Description:

Adds two expressions

Parameters:

Expression1:

Any valid TrioBASIC expression

Expression2:

Any valid TrioBASIC expression

Example:

Add 10 onto the expression in the parentheses (18.9) and store in a local variable ‘result’, which will then contain the value 28.9

result = 10 + (2.1 * 9)

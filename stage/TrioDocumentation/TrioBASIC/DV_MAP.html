DV_MAP

Type:

System Command

Syntax:

DV_MAP

Description:

Displays the current DV_IN and DV_OUT mapping to the command line terminal.  Open terminal #0 and type DV_MAP at the >> prompt.

Parameters:

None.

Examples:

>>DV_MAP

Device Variable map :
DV_IN map :

     0-  3 : ECAT Input Module @ Slot 0 (Auto Inc Address = 1)

     4-  7 : ECAT Input Module @ Slot 0 (Auto Inc Address = 2)

 

DV_OUT map :

     0-  3 : ECAT Output Module @ Slot 0 (Auto Inc Address = 1)

     4-  7 : ECAT Output Module @ Slot 0 (Auto Inc Address = 2)

See Also:

DV_CONTROLWORD , DV_IN , DV_OUT , DV_STATUSWORD

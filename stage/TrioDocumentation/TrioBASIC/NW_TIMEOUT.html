NW_TIMEOUT

Type:

System Array (MC_CONFIG)

Syntax:

NW_TIMEOUT(slot, param_index) = value

Description:

NW_TIMEOUT is a slot based system parameter used to define the EtherCAT network timeout parameters during network startup.  A timeout value can be set for each phase of the startup sequence or for the overall EtherCAT startup.

Parameters:

slot:

The slot index for the EtherCAT port.

param_index:

Identifies which timeout parameter to configure.

Index

Description

0

Overall ESM* startup timeout

1

Initial to pre-operational timeout

2

Pre-operational to Safe-operational timeout

3

Safe-operational to Operational timeout

value:

Timeout value.  (units depend upon parameter)

* ESM = EtherCAT State Machine.

Example:

A slave node requires 4 seconds to allow it time to go from Pre-operational to Safe-Operational state.  The EtherCAT network is running on Slot 0 of the Motion Coordinator.  Add the NW_TIMEOUT to MC_CONFIG:

NW_TIMEOUT(0, 2) = 4000

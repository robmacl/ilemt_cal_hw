When the Motion Coordinator is LOCKED, 1000 is added to the above numbers. For example a locked MC664 will return 1664.

Examples:

Example 1:

Checking the control value of a locked controller on the command line:

>>PRINT CONTROL

1464

>> 

Example 2:

Checking the controller type in a program, if it fails then stop the programs. :

IF CONTROL <> 664 THEN

  PRINT #terminal, “This program was designed to run a MC664”

  HALT

ENDIF

()  Array Indexing

Type:

Program Variable

Syntax:

variable(index-0[, index-1[, index-2]]])

Description:

Allows the elements of an array to be accessed by specifying the index or indices of the element required.

Arrays are declared in a DIM or STATIC statement by specifying the size of each dimension after the data-type. The dimensions are indexed from 0 to the dimension’s size minus 1.

Parameters:

variable:

the variable being indexed

index-0:

the index of the first dimension

index-1

the index of the second dimension

index-2

the index of the third dimension

Examples:

Example 1

Set element 0 of array ‘a’ to 0.0 and then display its value.

DIM a AS FLOAT ( 6 )

a( 0 ) = 0.0
PRINT "a(0) = " ; a( 0 )

Example 2

Display all the elements of a 2-dimensional array.

DIM a AS FLOAT ( 2 , 3 )
DIM i, j AS INTEGER

FOR i = 0 TO 1
     FOR j = 0 TO 2
         PRINT a(i, j);
         IF j < 2 THEN PRINT ", " ;
     NEXT j
     PRINT
NEXT i

Example 3

Set each row of a 2-dimensional array individually.

DIM a AS FLOAT ( 2 , 3 )

a( 0 ) = 1 , 2 , 3
a( 1 ) = 4 , 5 , 6

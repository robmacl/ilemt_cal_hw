AOUT_CONFIG

Type:

I/O Command

Syntax:

AOUT_CONFIG(channel) = value

Description:

Sets the mode of operation for an analogue output. This operation can be performed on any external analogue input from a Trio CAN Analogue input/output module P326 and P328.

This function requires CANIO firmware 2.3.2 or later.

The CAN analogue module P326/P328 has 4 x 12 bit outputs.

Parameters:

channel:

Analogue output channel number.

value:

0

Bipolar, -10 V .. +10 V

 

1

Unipolar, 0 .. +10 V

 

2

Bipolar, -5 V .. +5 V

 

3

Unipolar, 0 .. +5 V

Example:

Set analogue output 2 to range 0 to 10 V. The 12 bit value range will be 0 to 4095 where 4095 := 10V.

AOUT_CONFIG(2) = 1

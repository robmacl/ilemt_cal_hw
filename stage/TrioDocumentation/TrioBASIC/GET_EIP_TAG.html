GET_EIP_TAG

Type:

System Function

Syntax:

GET_EIP_TAG([name[, vr_index]])

Description:

Return information about the defined tags, and either print to the standard output stream or write into the VRs from the given index.

Note: only the display write to the command line (terminal #0) has been implemented in the current releases.

Parameters:

name:

Optionally included is the name of the tag (of data type string) whose configuration is to be returned. If there are no parameters or the name parameter is a null string, then return all defined tags.

vr_index:

Optional parameter to indicate where in the VRs the tag information will be written. If -1 (default), then write to the standard output stream.

See Note above.

Examples:

Example 1:

Display all tags.

>>GET_EIP_TAG                  

x_pos: Memory: 1 (Vr), Address: 100, Datatype: 0xC3

y_pos: Memory: 1 (Vr), Address: 101, Datatype: 0xC7

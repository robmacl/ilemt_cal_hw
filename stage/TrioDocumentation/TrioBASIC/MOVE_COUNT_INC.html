MOVE_COUNT_INC

Type:

Axis Parameter

Description:

MOVE_COUNT_INC defines the value that MOVE_COUNT will change every time a motion command is loaded and starts. The setting can be used when a pattern of moves is stopped and then reversed, for example in a machine following a series of points through an XY path.

Value:

1

MOVE_COUNT = MOVE_COUNT + 1 for each new move

100

MOVE_COUNT = MOVE_COUNT + 100 for each new move

-1

MOVE_COUNT = MOVE_COUNT - 1 for each new move

-50

MOVE_COUNT = MOVE_COUNT - 50 for each new move

0

MOVE_COUNT will not change after each new move

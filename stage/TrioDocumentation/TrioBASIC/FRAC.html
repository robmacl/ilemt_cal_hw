FRAC

Type:

Mathematical Function

Syntax:

value = FRAC(expression)

Description:

Returns the fractional part of the floating-point value.

Parameters:

value:

The fractional part of the expression

expression:

Any valid TrioBASIC expression

Example:

Print the fractional part of 1.234 on the command line

>>PRINT FRAC(1.234)

0.2340

>>

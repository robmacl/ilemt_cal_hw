GLOBAL

Type:

System Command

Syntax:

GLOBAL ["name"[, vr_number]]

Description:

GLOBALs can be declared in the controller which are available to all programs. GLOBAL declares the name as a reference to one of the global VR variables. The name can then be used both within the program containing the GLOBAL definition and all other programs in the Motion Coordinator project.

They must be declared on startup.  The program declaring GLOBALs must be the ONLY process running at power-up.

Once a GLOBAL has been assigned it cannot be changed, even if you change the program that assigns it.

Model number

Max number of GLOBALs

MC464

1024

MC403(Z), MC405, Euro404, Euro408, MC4N-ECAT, MC4N-RTEX

4096

MC664(X), MC508, Flex-6 Nano, Flex-6X Nano, MC6N-ECAT

8192

While developing you may wish to clear or change a GLOBAL. You can clear a single GLOBAL by using the first parameter alone. All GLOBALs can be cleared by issuing GLOBAL. You can view all GLOBALS using LIST_GLOBAL .

Whenever a program is RUN, the GLOBAL and CONSTANT tables are searched for the names of already defined variables.  If a large number of GLOBAL/CONSTANT names are pre-defined, this may take several seconds.  When using the GLOBAL and CONSTANT system is it not advised to depend on temporarily running and then stopping programs.  It is best to run all the required programs when the machine starts and leave them in that running state.

Parameters:

name:

Any user-defined name containing lower case alpha, numerical or underscore (_) characters.

vr_number:

The number of the VR to be associated with name.

Example:

Example 1:

Initialise two GLOBALs and use then to adjust machine parameters.

GLOBAL "screw_pitch",12

GLOBAL "ratio1",534

 

ratio1 = 3.56

screw_pitch = 23.0

PRINT screw_pitch, ratio1

Example 2:

Initialise 3 GLOBALs, delete one of the GLOBALs, and then delete all GLOBALs.

>>GLOBAL "pitch",12
>>GLOBAL "roll",534
>>GLOBAL "yaw",922
>>LIST_GLOBAL ‘ Display GLOBALs
Global                           VR
----------------                 ----
pitch                            12
roll                             534
yaw                              922
Constant                         Value
----------------                 -------

 

>>GLOBAL "roll" ‘ Delete roll
>>LIST_GLOBAL ‘ Display GLOBALs
Global                           VR
----------------                 ----
pitch                            12
yaw                              922
Constant                         Value
----------------                 -------

 

>>GLOBAL ‘ Delete all GLOBALs
>>LIST_GLOBAL ‘ Display GLOBALs
Global                           VR
----------------                 ----
Constant                         Value
----------------                 -------

>> 

Example 3:

The program to initialise GLOBAL and CONSTANT values runs on power up using the default process number.  Once finished, it then runs the main program to start machine operations.

' machine settings

GLOBAL "initial_speed", 10

GLOBAL "run_speed", 11

' vision data via Modbus

GLOBAL "vision_x", 51

GLOBAL "vision_y", 51

GLOBAL "vision_r", 52

GLOBAL "vision_status", 53

 

CONSTANT "x_axis", 0

CONSTANT "y_axis", 1

CONSTANT "v_status_ok", 8

CONSTANT "v_status_error", 4

 

RUN "MAIN", 2 ' run the main program on process #2

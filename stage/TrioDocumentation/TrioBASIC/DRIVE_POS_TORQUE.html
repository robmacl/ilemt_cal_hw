DRIVE_POS_TORQUE

Type:

Axis Parameter

Description:

Sets the positive torque limit value in a remote drive.  For this value to be active, the cyclic data transfer from the drive must be first configured to set the POSITIVE_TORQUE_LIMIT value.  For a drive connected by CANopen over EtherCAT (CoE) the value will be configured as part of the Process Data Object (PDO).

Usually, the parameter is mapped to Object 0x60E0 sub index 0x00, size 16 bits.

Value:

The drive positive torque limit is an INT16 in drive units, often it is percent x 10.  See the drive’s manual for details.

Example:

Example1

Set the drive’s positive torque limit to 10% from a program.

DRIVE_POS_TORQUE AXIS(2) = 100

Example2

Reduce the drive’s positive torque limit to 5% after the position is close to 4 mm from end of travel.

BASE(1)

DRIVE_POS_TORQUE = 1000

MOVE(30)
WAIT UNTIL REMAIN < 4

DRIVE_POS_TORQUE = 50

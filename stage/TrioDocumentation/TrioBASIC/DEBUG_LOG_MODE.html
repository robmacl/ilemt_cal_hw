DEBUG_LOG_MODE

Type:

System Parameter (MC_CONFIG)

Description:

This parameter enables debug logging.  Information is saved in the debug log in the flash memory and can be retrieved by typing DEBUG_LOG.

Value:

Bit

Description

Value

1

Enable Modbus status and error logging.

2

Examples:

Example 1:

Enable Modbus error logging.

DEBUG_LOG_MODE.1 = 1

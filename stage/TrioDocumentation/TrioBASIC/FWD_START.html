FWD_START

Type:

Axis Parameter

Description:

Force an immediate forward jog motion.   When FWD_JOG is -1 and FWD_START is set to 1, the axis will run the MTYPE 14 forward_jog in the axis.  If FWD_JOG is set to an input number, (not equal to -1) then FWD_START has no function.

Value:

FWD_START

FWD_JOG

MTYPE

0

-1

0 (Idle)

1

-1

14 (forward_jog)

Not Equ 1

-1

0 (Idle)

Any

Not Equ -1

0 (Idle) *

* Under control of the designated input

Example:

BASE(2)

FWD_JOG = -1

REV_JOG = -1

 

IF IDLE THEN

  REPEAT

    SELECT_CASE VR(10)

    CASE 0

      FWD_START = 0

      REV_START = 0

    CASE 1

      FWD_START = 1

      REV_START = 0

    CASE 2
      FWD_START = 0

      REV_START = 1

    END_CASE

  UNTIL VR(10) = 99
ENDIF

FLASH_STATUS

Type:

System Parameter

Syntax:

value = FLASH_STATUS

Description:

This read-only parameter returns the status of the flash memory.

The Flash memory can be busy when programs have been edited or new programs downloaded. Flash re-programming is done as a background process and large files take some time to store.

On Flex-7, flash read/write time is fixed regardless of the number of VRs written, unlike other controllers where it increases with data size.

Parameters:

Bit

Description

0

1 = flash busy (data may be lost if the controller is powered off)
0 = flash idle (data saved, so safe to power off the controller)

Examples:

Example 1:

PRINT FLASH_STATUS

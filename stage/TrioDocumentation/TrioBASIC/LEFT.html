LEFT

Type:

String Function

Syntax:

value = LEFT(string, length)

Description:

Returns the left most section of the specified string using the length specified.

Parameters:

string:

String to be used

length:

Length of string to be returned

Examples:

Example 1:

Pre-define a variable of type string and later print its leftmost 4 characters:

DIM str1 AS STRING(32)

str1 = "TRIO MOTION TECHNOLOGY"

PRINT LEFT(str1, 4)

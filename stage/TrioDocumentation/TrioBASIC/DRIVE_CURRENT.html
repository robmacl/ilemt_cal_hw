DRIVE_CURRENT

Type:

Axis Parameter

Description:

Returns the actual current value from a remote drive. For this value to be active, the cyclic data transfer from the drive must be first configured to return the drive actual current value. For a drive connected by CANopen over EtherCAT (CoE) the value will be configured as part of the Process Data Object (PDO).

Value:

The drive current returned in drive units.

Example:

Example1

Display the motor current to Motion Perfect terminal 5.

PRINT #5,"Drive current value = "; DRIVE_CURRENT AXIS(2)

Example2

Wait for the motor current value to go below a pre-defined level.

BASE(16)

WAIT UNTIL DRIVE_CURRENT < 500

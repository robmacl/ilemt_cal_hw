IS_OPEN

Type:

File Function (Read Only)

Syntax:

value = IS_OPEN #channel

Description:

IS_OPEN returns the state of the selected channel. The channel may be a communications port, a file or a temporary buffer. IS_OPEN state TRUE does not imply that the Ethernet connection is valid and open, it only refers to the channel in the controller firmware so as to find out if a channel number is already in use.

Parameters:

value:

TRUE (-1) if channel is open,
FALSE (0) if the channel is closed.

channel:

The channel number using the # channel notation

Examples:

Example 1:

DIM comm_status AS BOOLEAN


comm_status = IS_OPEN #20

Example 2:

Check the status of the TCP/IP connection on channel 21 and if it is available, then open it as a TCP client on port 1000.

IF FALSE = IS_OPEN #21 THEN

    OPEN #21 AS "TCP:192.168.0.230(1000)" FOR READ_WRITE

    WA(20)

ENDIF

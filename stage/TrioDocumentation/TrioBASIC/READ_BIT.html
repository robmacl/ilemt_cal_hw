READ_BIT

Type:

Logical and Bitwise operator

Syntax:

READ_BIT(bit, index [, mode] )

Description:

READ_BIT can be used to test the value of a single bit within a VR or TABLE element.

By default, VR will be used, but the mode parameter can be changed to operate on TABLE data.

Parameters:

bit:

The bit number to clear, valid range is 0 to 52

variable:

The VR / TABLE which to operate on

Mode:

0

Bit reads from VR (default)

 

1

Bit reads from TABLE

Examples:

Example 1:

Read bit 4 o f VR (13).

result = READ_BIT(4, 13)

Example 2:

Read bit 4 of  TABLE (13)

result = READ_BIT(4, 13, 1)

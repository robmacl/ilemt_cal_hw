TAN

Type:

Mathematical Function

Syntax:

value = TAN(expression)

Description:

Returns the natural tangent of the supplied angle measured in radians.

Parameters:

value:

The natural tangent of the expression

expression:

Any valid TrioBASIC numeric expression representing an angle measured in radians

Example:

Print the natural tangent of 1 radian on the command line

>>PRINT TAN(1)

1.5574

>>

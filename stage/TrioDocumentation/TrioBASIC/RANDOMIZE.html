RANDOMIZE

Type:

Process Command

Syntax:

RANDOMIZE

RANDOMIZE(seed)

Description:

This command initialises the pseudo-random number generator for the process it is issued within (so issuing this command in one process will not influence the pseudo-random sequence generated in another process).

Without parameters, the pseudo-random number generator is primed with a seed that is generated from the constantly varying internal system state.

With a parameter, the pseudo-random number generator is primed with the seed specified.

Note that use of this command is not normally required, as each time a program is executed, it implicitly calls RANDOMIZE with no parameters to ensure that the sequence is generated from an unknown starting point. However, if you require a known series of pseudo-random values, for example when debugging or testing, then you can use this command with a parameter, usually zero, before you call RND or RNDF.

Parameters:

seed:

The integer value that the pseudo-random number generator will use to as the base from which to generate values from

Examples:

Example 1:

Print some random numbers on the command line with and without RANDOMIZE

>>PRINT RND, RND, RND

3337245933     325304933     1294402698

>>RANDOMIZE

>>PRINT RND, RND, RND

447495379     435546399     1826854126

>>RANDOMIZE(0)

>>PRINT RND, RND, RND

2627695341     1250387728     1117271038

>>PRINT RND, RND, RND

782908658     3895116732     3594165340

>>RANDOMIZE(0)

>>PRINT RND, RND, RND

2627695341     1250387728     1117271038

>>PRINT RND, RND, RND

782908658     3895116732     3594165340

>>

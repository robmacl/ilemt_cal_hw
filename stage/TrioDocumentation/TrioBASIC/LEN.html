LEN

Type:

String Function

Syntax:

value = LEN(string)

Description:

Returns number of characters in the specified string.

Note that this will differ from the number of bytes used to store the string when UTF-8 characters are present.

Parameters:

string:

String to be measured.

Examples:

Example 1:

Define a variable of type string, assign to it, and then display its length.

DIM str1 AS STRING(20)

str1 = "MyString"

x = LEN(str1) 'x will be 8

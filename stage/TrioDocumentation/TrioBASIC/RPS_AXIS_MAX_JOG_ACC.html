AXIS_MAX_JOG_ACC

Type:

Robotics Parameter

Description:

Sets the axis maximum jog acceleration in user AXIS_UNITS / sec 2 for a robot axis can run at when performing a jog.

Value:

The acceleration in user AXIS_UNITS / sec 2 which an axis will use when being jogged.

See Also:

AXIS_JOGSPEED , AXIS_UNITS

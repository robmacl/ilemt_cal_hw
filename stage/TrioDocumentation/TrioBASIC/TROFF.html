TROFF

Type:

System Command

Syntax:

TROFF [“program”]

Description:

The trace off command resumes execution of the SELECT ed or specified program. The command can be included in a program to resume the execution of that program.

For de-bugging the Motion Perfect breakpoint tool should be used.

Parameters:

program:

The name of the program which you wish to resume

Example:

Resume execution of a program names TEST

>>TROFF "TEST"

OK

>>%[Process 21:Program TEST] – Released

See Also:

HALT , STOP , STEPLINE , TRON

PROG_TYPE

Type:

System Parameter (Read Only)

Syntax:

value = PROG_TYPE "file_name"

Description:

The PROG_TYPE system parameter returns the type of a file. If the file does not exist, it returns -1.

Value:

The following PROG_TYPE’s are currently active values.

Value

Description

0

BASIC

1

Encrypted BASIC

2

GCode

3

TEXT

4

IEC

5

FIFO

6

TEMP

7

Project

8

Binary

9

MC Config

10

IEC Library

11

HMI Design

12

BASIC Library

13

BASIC Library encrypted

14

HMI Library

15

Volatile Text

16

Robot

17

Robot Basic

-1

File does not exist

Examples:

Example 1:

If a file a program exists and it is a TrioBASIC file then run it:

 

triobasic_type = 0

IF IS_PROG "MAIN_TASK" THEN

     IF (PROG_TYPE "MAIN_TASK") = triobasic_type THEN

     RUN "MAIN_TASK"

ENDIF

ENDIF

DAC_OUT

Type:

Axis Parameter (Read Only)

Description:

This parameter returns the demand value for the axis.

In an analogue system this will be the value sent to the voltage output (the D to A converter). If SERVO = ON this is the output of the closed loop algorithm. If SERVO = OFF it is the value set by the user in DAC . DAC_OUT value, and hence the actual output is limited by the setting in OUTLIMIT .

In a digital system DAC_OUT returns the demand value for the axis which could be the Target Position, Target Speed or Target Torque depending on the axis ATYPE .

Value:

Demand value for the axis.

Examples:

Example 1:

To check that the controller has set the correct voltage for axis 8 on an analogue system read DAC_OUT in the command line.

>>PRINT DAC_OUT AXIS(8)

288.0000

>> 

Example 2:

Check that Target Position of an EtherCAT CSP axis is being received by the drive. Read DAC_OUT to see the value in the PDO and object 0x607A to see the target position value in the drive.

ax = 2
PRINT "Target position in PDO = "; DAC_OUT AXIS(ax)
CO_READ_AXIS(ax, $607A, 0, 4, 20)

PRINT "Target position in drive = "; VR(20)

Example 2:

Check that Target Velocity of an EtherCAT CSV axis is being received by the drive. Read DAC_OUT to see the value in the PDO and object 0x60FF to see the target velocity value in the drive.

ax = 2
PRINT "Target velocity in PDO = "; DAC_OUT AXIS(ax)
CO_READ_AXIS(ax, $60FF, 0, 4, 20)

PRINT "Target velocity in drive = "; VR(20)

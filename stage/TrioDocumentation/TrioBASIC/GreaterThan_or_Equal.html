>=  Greater Than or Equal

Type:

Comparison Operator

Syntax

<expression-1> >= <expression-2>

Description:

Returns TRUE if <expression-1> is greater than or equal to <expression-2>, otherwise returns FALSE .

The comparison operators are typically used in conditional statements like IF , REPEAT and WHILE , though can be used anywhere an expression is valid.

Parameters:

Expression1:

Any valid TrioBASIC expression

Expression2:

Any valid TrioBASIC expression

Example:

If variable ‘target’ holds a value greater than or equal to 120, then move to the absolute position of 0.

IF target >= 120 THEN MOVEABS(0)

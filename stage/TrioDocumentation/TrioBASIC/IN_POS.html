IN_POS

Type:

Axis Parameter

Syntax:

<value> = IN_POS

<value> = IN_POS AXIS(n)

Description:

This axis parameter signals that the axis move has finished and that the axis is in position within defined limits of following error and speed. The IN_POS feature simplifies the checking required at end of moves to ensure a motor has reached position.

Using IN_POS requires parameters IN_POS_DIST and IN_POS_SPEED to be set up first.

The IN_POS_DIST and IN_POS_SPEED default to zero. Reaching zero position and speed error may take considerable time or not happen, depending on the servo configuration.

Value:

Returns TRUE when the following 4 conditions are all true:

- move completed (IDLE completed)

- no moves are buffered (i.e. NTYPE and PMOVE are zero)

- following error FE is less than IN_POS_DIST

- measured speed MSPEED is less than IN_POS_SPEED

Example:

Make a move and wait for the axis to be in position before continuing.

BASE(0)
SERVO = ON
UNITS = 100000
SPEED = 5
ACCEL = 50
DECEL = 50

IN_POS_DIST = 0.002
IN_POS_SPEED = 0.1
REPEAT
  TRIGGER
  MOVE(1)
  WAIT IN_POS
  WA(50)
UNTIL FALSE

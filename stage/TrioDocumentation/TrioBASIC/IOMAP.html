IOMAP

Type:

I/O Command (command line only)

Syntax:

IOMAP[(mode)]

Description:

Lists the current digital I/O map, analogue I/O map or full I/O map.

Parameters:

mode

0

Show digital I/O only (Default)

1

Show analogue I/O only

2

Show digital and analogue I/O

Example1:

>> IOMAP

Digital Input map :
     0-   7 : Built-in Inputs
     8-  15 : Built-in Bi-Directional IO
    16-  31 : CAN P318 @ Address 0 (fw=v1.3.0)
    32-1023 : Virtual

Digital Output map :
     0-   7 : Virtual
     8-  15 : Built-in Bi-Directional IO
    16-  31 : CAN P327 @ Address 0 (fw=v1.3.0)
    32-1023 : Virtual

Example2:

Show analogue I/O connected using CANopen

>>IOMAP(1)

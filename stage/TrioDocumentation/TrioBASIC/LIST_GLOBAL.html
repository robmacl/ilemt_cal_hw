LIST_GLOBAL

Type:

System Command (command line only)

Syntax:

LIST_GLOBAL [offset[, length]]

Description:

Prints the selected GLOBAL and CONSTANT definitions to the current output channel.

Parameters:

none:

Prints all the defined GLOBAL and CONSTANT values.

offset:

Number of the first GLOBAL or CONSTANT value to be printed.

length:

Number of GLOBAL and CONSTANT values to be printed. If length is -1 then all values are printed.

Example:

Check all global data in an application where the following GLOBAL and CONSTANT have been set.

CONSTANT "cutter", 23

GLOBAL "conveyor",5

 

>>LIST_GLOBAL

Global                           VR

----------------                 ----

conveyor                         5

Constant                         Value

----------------                 -------

cutter                           23.00000

>>LIST_GLOBAL 1

Global                           VR

----------------                 ----

conveyor                         5

>>LIST_GLOBAL 0,1

Constant                         Value

----------------                 -------

cutter                           23.00000

>>

REP_OPTION

Type:

Axis Parameter

Description:

REP_OPTION allows different repeat options for the axis. It can be used to affect the way the position of an axis wraps or the repeating mode of CAMBOX , MOVELINK and FLEXLINK .

Value:

Bit

Description

Value

0

0

Axis position range is - REP_DIST to + REP_DIST

1

1

Axis position range is 0 to + REP_DIST

1

 

0

Automatic repeat option is enabled

2

1

Disable the automatic repeat option  of CAMBOX and MOVELINK

2

0

REP_DIST , DEFPOS and OFFPOS will affect MPOS and DPOS

4

1

REP_DIST , DEFPOS and OFFPOS will affect MPOS only

3

0

FRAME_REP_DIST is disabled

8

1

This mode is to be used with FRAME and USER_FRAME only and has the following functionality:

REP_DIST is disabled

FRAME_REP_DIST is used when FRAME <> 0 or USER_FRAME <> 0

FRAME_REP_DIST will only change DPOS and WORLD_DPOS

DATUM , DEFPOS and OFFPOS only work when FRAME = 0 and USER_FRAME (0)

4

0

Default REP_DIST behaviour

16

1

This mode is applied only when bit 3 is 0 - REP_DIST is applied to MPOS , AXIS_DPOS and DEMAND_EDGES , but not DPOS . This mode is useful when a FRAME is active.

5

0

DRIVE_REP_DIST will be applied when drive position ( DAC_OUT ) is greater than this value

32

1

DRIVE_REP_DIST will be applied when drive position ( DAC_OUT ) is greater than or equal to this value.

Bit 2 has been included for backward compatibility; it is not recommended to use this in new applications.

Examples:

Example 1:

An axis has 4000 counts per revolution, configure REP_DIST and REP_OPTION so that it wraps from 0 to 4000.

REP_OPTION.0 = 1

REP_DIST = 4000

Example 2:

A program is running a continuous MOVELINK , when an input is triggered the link must end at the end of the next cycle. Set only bit 1, do not clear any other bits that may be active.

MOVELINK(1, 1.6, 0.6, 0.6, 1, 4)

WAIT UNTIL IN(1) = ON

REP_OPTION.1 = 1

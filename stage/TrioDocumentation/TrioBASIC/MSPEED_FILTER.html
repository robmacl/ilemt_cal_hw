MSPEED_FILTER

Type:

Axis Parameter

Description:

MPSEED_FILTER is the filter coefficient value for use by MSPEEDF . MSPEED_FILTER can take any value between 0 and 1.0. Lower values mean longer time constant and therefore greater filtering effect. When set to 1, the MSPEEDF value will be the same as MSPEED .

The filter equation approximates to the following and is processed once per SERVO_PERIOD .

MSPEEDF = (1 - MSPEED_FILTER) * MSPEEDF + MSPEED_FILTER * MSPEED

Value:

Filter time constant coefficient, which may take a value between 0 and 1 (default).

Example:

Monitor MSPEEDF to detect when the speed of axis 3 rises above a threshold value.

BASE(3)

MSPEED_FILTER = 0.5 'Initialise the filter coefficient

 

WHILE TRUE

  IF MSPEEDF > 5 THEN

    GOSUB start_main

  ENDIF

  ...

WEND

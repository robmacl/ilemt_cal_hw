VIEW

Type:

System Command

Syntax

VIEW "program_name"

Description:

Display source and object code data to the terminal.  Optional string parameter can be used to view a specific program otherwise the currently selected program will be used, eg VIEW "STARTUP".

The VIEW command will not show the memory contents if the controller is LOCK ed or the project programs are Encrypted.

Example:

>>VIEW "D2"

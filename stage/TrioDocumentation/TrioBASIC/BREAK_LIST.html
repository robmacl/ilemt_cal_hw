BREAK_LIST

Type:

System Command (command line only)

Syntax:

BREAK_LIST "program name"

Description:

Displays a list of all the breakpoints in the given program name. The output consists of the program name followed by each breakpoint’s line number and the code on that line.

Primarily for use by Motion Perfect.

Parameters:

program name:

the name of any program existing on your controller

Example:

Show the breakpoints from a program called "simpletest" that has breakpoints inserted on lines 8 and 11.

>>BREAK_LIST "simpletest"

 

Program: SIMPLETEST

Line 8: SERVO = ON

Line 11: BASE(0)

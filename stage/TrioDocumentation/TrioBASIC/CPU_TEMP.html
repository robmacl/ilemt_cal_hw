CPU_TEMP

Type:

System Parameter (read only)

Syntax:

value = CPU_TEMP

Description:

Read the CPU’s temperature in degrees Celsius (°C).

CPU_TEMP is not available for Euro404/408, MC403, MC403-Z, MC405, MC405-Y, MC4NE and MC4N-RTEX.

Example:

Display the CPU’s temperature on the command line.

PRINT CPU_TEMP

Will display:

52.6

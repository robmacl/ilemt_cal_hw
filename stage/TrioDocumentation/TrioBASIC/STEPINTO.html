STEPINTO

Type:

System Command

Syntax:

STEPINTO ["program"[, process]]

Description:

Steps one line in a program, but unlike the STEPLINE command, steps into any functions encountered rather than over them. This command is usually issued from the command line, but it can also be used as a statement within a program, in which case it will cause execution to pause when encountered.

This command is also used by Motion Perfect to control program stepping. 

All copies of this named program will step unless the process number is also specified.

If the program is not running it will step into either the specified process or the next available process if the process parameter is omitted, pausing execution at the first executable line.

If the program name is not supplied, either the SELECT ed program will step (if used from the command line), or the program that paused due to an embedded STEPINTO statement will step.

Parameters:

program:

This specifies the program to be stepped.

process:

Specifies the process number.

Example:

Start the program conveyor running in the highest available process by stepping into the first executable line.

>>STEPINTO "conveyor"              

OK

%[Process 21:Line 19] – Paused

>>

RIGHT

Type:

String Function

Syntax:

value = RIGHT(string, length)

Description:

Returns the rightmost section of the specified string using the length specified.

Parameters:

string:

String to be used

length:

Length of string to be returned

Examples:

Example 1:

Pre-define a variable of type string and later print its right most 10 characters:

DIM str1 AS STRING(32)

str1 = "TRIO MOTION TECHNOLOGY"

PRINT RIGHT(str1, 10) 'Will display "TECHNOLOGY"

PCMCAT_PROC

Type:

System Parameter (MC_CONFIG / FLASH)

Description:

When the PC-MCAT API opens a connection to the Motion Coordinator, the PCMCAT_PROGRAM is started on the highest available process.

PCMCAT_PROC can be set to specify a different process for the PCMCAT_PROGRAM. If the defined process is in use then the next lower available process will be used.

This programs handles operations that cannot be handled through direct access to shared memory. A FIFO is initialised in shared memory when the API opens a connection to handle these operations. The PCMCAT_PROGRAM would manage this FIFO acting as a bridge between the shared memory and the Motion Coordinator.

PCMCAT_PROC is stored in Flash EPROM and can also be set in the MC_CONFIG script file.

This command only works on the PC-MCAT and PC-MCAT S products.

Value:

-1

Use the highest available process (default)

0 to max process

Run on defined process

Examples:

Example1:

Set PCMCAT_PROGRAM to start on process 19 or lower (using the command line terminal).

>>PCMCAT_PROC = 19

>> 

Example2:

Remove the PCMCAT_PROC setting so that PCMCAT_PROGRAM starts on default process (using MC_CONFIG).

'MC_CONFIG script file

PCMCAT_PROC = -1  'Start on default process on connection

SYNC_FS_LIMIT

Type:

Robotics Parameter

Description:

An end of travel limit may be set up in software thus allowing synchronization to stop when the robot is going outside the limit. The robot will wait with SYNC_CONTROL = -1 if the controller is waiting for the synchronization point to move within range.

Value:

The absolute position of the software forward travel limit in user UNITS for Robot Synchronization.

Examples:

Example 1:

Define the forward synchronization limit to 100 units. The robot will start synchronization when the target point position on axis 0 is less than  SYNC_FS_LIMIT. SYNC_CONTROL will be -1 before starting synchronization. Synchronization will be stopped if the robot end effector travels outside the limits while in sync.

BASE (0)

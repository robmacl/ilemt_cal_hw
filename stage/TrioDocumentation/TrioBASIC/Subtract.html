-  Subtract

Type:

Mathematical Operator

Syntax:

<expression1> - <expression2>

Description:

Subtracts expression2 from expression1

Parameters:

Expression1:

Any valid TrioBASIC expression

Expression2:

Any valid TrioBASIC expression

Example:

Evaluate 2.1 multiply by 9 and subtract the result from 10, this will then be stored in VR 0, leaving it containing the value -8.9

VR(0) = 10 - (2.1 * 9)

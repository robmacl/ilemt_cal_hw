ROBOT_ERROR

Type:

Robotics Parameter (Read Only)

Syntax:

ROBOT_ERROR(group)

Description:

Returns the errors present in the RPS 2.0 kernel. There may be multiple errors present, with the value in the table below showing what each bit set represents.

ROBOT_ERROR requires the RPS runtime FEC

RPS 2.0 kernel must be activated using the RPS_KERNEL_V2 keyword to enable its use.

Bit

Value

Error

1

Examples:

Example 1:

Detect an emergency stop activation in Robot(0).

IF (ROBOT_ERROR(0) AND 256) <> 0 THEN
   ‘Emergency stop has been triggered
ENDIF

IOSIZE

Type:

System Parameter (Read Only)

Description:

Returns the total number of digital I/O channels, both real and virtual, which varies across controller types.

The digital I/O is numbered from zero, so the valid range is 0 to IOSIZE - 1.

Value:

The maximum number of digital I/O channels.

Example:

Turn on the last digital output.

>>OP(IOSIZE – 1, ON)

>>

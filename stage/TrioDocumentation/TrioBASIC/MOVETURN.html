MOVETURN

Type:

Axis Command

Syntax:

MOVETURN(mode, axes, x_start, y_start, x_end, y_end, sp_start, sp_end[, acc[, x_po[, y_po[, sp_lim]]]])

MOVETURN(mode, axes, x_start, y_start, z_start, x_end, y_end, z_end[, sp_start[, x_po[, y_po[, sp_lim]]]])

Description:

Parameters:

mode:

0

 

1, 2

 

3

 

4

Equivalent of mode 1, but for 3 axes

parameters:

Varies based on mode (see below)

mode = 0

MOVETURN performs an exterior change of direction.

MOVETURN(mode, axes, x_start, y_start, x_end, y_end, sp_start, sp_end[, acc])

Parameters:

mode:

0

axes:

Number of axes (must be 2 for this mode)

x_start:

Entry vector x component

y_start:

Entry vector y component

x_end:

Exit vector x component

y_end:

Exit vector y component

sp_start:

Entry speed of the MOVETURN

sp_end:

Exit speed of the MOVETURN

acc:

Acceleration/deceleration rate to use (Optional)

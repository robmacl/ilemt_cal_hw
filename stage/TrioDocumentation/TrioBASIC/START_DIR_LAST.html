START_DIR_LAST

Type:

Axis Parameter (Read Only)

Description:

Returns the direction in radians of the start of the last loaded advanced speed control interpolated motion command (those suffixed “SP”).

The direction is that of the first two axes of the move loaded, the direction being relative to the Y axis, so 0 is up the Y axis, PI/2 is up X axis, +PI and -PI are down the Y axis, and -PI/2 is down the X axis.

Note that this parameter is relevant to SP motion commands only, such as MOVESP , MOVEABSSP etc.

Value:

The move’s start direction in radians between - PI and PI .

Example:

Run two moves the first starting at a direction of 45 degrees and the second 0 degrees.

>>MOVESP(10000, 10000)

>>PRINT START_DIR_LAST

0.7854

>>MOVESP(0, 10000)

>>PRINT START_DIR_LAST

0.0000

>>

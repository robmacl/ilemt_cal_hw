MSPEED

Type:

Axis Parameter (Read Only)

Description:

MSPEED can be used to represent the speed measured as it represents the change in measured position in user UNITS (per second) in the last servo period.

This value represents a snapshot of the speed and significant fluctuations can occur, particularly at low speeds. MPSEEDF can be used to obtain a filtered version of MSPEED.

Value:

Change in measured position per second in user UNITS .

Example:

Check MSPEED in the command line terminal for axis 4.

>>BASE(4)

>>? MSPEED

12000.0000

>> 

MSPEED is change in MPOS per servo period * 1E6 / SERVO_PERIOD. This gives a scale of change in MPOS per second but does not use 1 second of data.

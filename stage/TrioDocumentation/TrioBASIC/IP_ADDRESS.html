IP_ADDRESS

Type:

System Parameter (MC_CONFIG / FLASH)

Description:

IP_ADDRESS is used to set the Ethernet IPv4 address of the main Ethernet port of the Motion Coordinator. This parameter uses the standard dot-decimal notation to define the 4 separate octets of the IP address that define the IP address’s 32-bit value.

The value is held in flash EPROM and can be set in the MC_CONFIG script.

Value:

Network IP address in dot-decimal notation, which consists of four decimal numbers separated by dots, where each number is an 8-bit value (an octet), and so has the range 0 to 255.

Examples:

Example 1:

IP_ADDRESS = 192.168.0.250

Example 2:

Set IP address in the MC_CONFIG file:

'MC_CONFIG script file

IP_ADDRESS = 192.168.2.100

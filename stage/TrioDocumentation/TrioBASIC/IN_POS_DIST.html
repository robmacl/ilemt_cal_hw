IN_POS_DIST

Type:

Axis Parameter

Syntax:

IN_POS_DIST AXIS(n) = <value>

Description:

IN_POS_DIST is a distance, scaled using the axis UNITS, indicating the required position error before IN_POS signals that the axis move has finished and that the axis is in position.

IN_POS_DIST defaults to zero. Reaching zero position and speed error may take considerable time or not happen if this parameter is not set.

Value:

The following error value in the units set for the axis.

Example:

Make a move and wait for the axis to be in position before continuing.

BASE(0)
SERVO=ON
UNITS=100000
SPEED=5
ACCEL=50
DECEL=50

IN_POS_DIST=0.002
IN_POS_SPEED=0.1
REPEAT
TRIGGER
  MOVE(1)
  WAIT IN_POS
  WA(50)
UNTIL FALSE

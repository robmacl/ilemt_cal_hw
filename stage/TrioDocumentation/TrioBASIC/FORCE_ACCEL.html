FORCE_ACCEL

Type:

Axis Parameter

Description:

This parameter sets the acceleration rate for a motion command that supports the advanced speed control. The VP_SPEED will accelerate at the rate given by FORCE_ACCEL.

With VP_MODE = 5 and VP_MODE = 6 with VP_OPTIONS.1 = 1, the forced demands (those loaded with the move) are bound to the axis demands such that the axis demands are primed with the forced demands of a move when it becomes active. This binding allows the axis demands to be adjusted, and the changes made influence the current move’s profiling.

FORCE_ACCEL is also used to provide the acceleration demand to MOVEADD.

Value:

The acceleration rate at which the motion command will execute, in user UNITS per second per second. The default value is -1 and means the parameter is not used.

Examples:

Example 1:

With continuous motion running on the axis, make an additional move to advance the axis by 40 mm after setting the speed, acceleration, deceleration and jerk values for the added movement.

FORWARD

WA(100)

 

FORCE_SPEED = 50

FORCE_ACCEL = 2000

FORCE_DECEL = 2000

FORCE_JERK = 50000

MOVEADD(40)

INTEGER_WRITE

Type:

Mathematical Function

Syntax:

INTEGER_WRITE(destination, least_significant, most_significant)

Description:

INTEGER_WRITE performs a low-level write to a 64-bit destination by combining the bit patterns of two 32-bit values into a 64-bit bit pattern and writing this into the destination. The destination may be integer or floating-point, as it is the bit pattern (i.e. internal representation) that is written, not its value or a cast of its value.

Parameters:

destination:

64-bit value that will be written, can be VR , TABLE , or system variable.

least_significant:

Least significant (rightmost) 32 bits, can be any valid TrioBASIC expression.

most_significant:

Most significant (leftmost) 32 bits, can be any valid TrioBASIC expression.

Examples:

Example 1

Write a 64-bit integer bit pattern to a VR and then read it back for display on terminal 0.

INTEGER_WRITE(VR(1), $1234, $5678)

 

INTEGER_READ(VR(1), VR(4), VR(5))

PRINT HEX(VR(4)), HEX(VR(5))

Example 2

Write the 64-bit pattern for the floating-point number 0.9999 and display as a float.

INTEGER_WRITE(VR(1), $48E8A71E, $3FEFFF2E)

PRINT VR(1)

AXIS_Z_OUTPUT

Type:

Axis Parameter

Description:

Controls the operation of the Z pins when used as outputs.  For example a Pulse + Direction axis with enable output uses the pins of the axis that are normally allocated to the Z encoder input as the Enable output.  This only applies to products where the line driver is bi-directional and can be input or output.

Value:

To be advised.

Example:

Allow AXIS_ENABLE to control the enable/Z output.

BASE(axis_number)

ATYPE = 77 ' Servo with incremental encoder and enable output

AXIS_Z_OUTPUT = 0 ' do not force the output, allow AXIS_ENABLE control

AXIS_ENABLE = ON ' Enable the axis

WDOG = ON ' enable all axes and set wdog relay to ON

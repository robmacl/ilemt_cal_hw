INVERT_IN

Type:

I/O Command

Syntax:

value = INVERT_IN(input)

INVERT_IN(input, state)

Description:

The INVERT_IN command allows digital input channels to be individually inverted in software.

This is important as these input channels can be assigned to activate functions such as feed-hold.

When called with one parameter, the current inversion state of the specified input number is returned (either ON or OFF).

When called with two parameters, the inversion state is set to that specified in the second parameter. The two-parameter form can also be used as a function, but it always returns TRUE (-1).

Parameters:

input:

The input to invert

state:

ON

the input is inverted in software

OFF

the input is not inverted

Example:

Invert input 7 so that when the input is low the FWD_JOG is not active.

INVERT_IN(7, ON)

FWD_JOG = 7

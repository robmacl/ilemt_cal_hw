DV_IN

Type:

System Command (Read Only)

Syntax:

value = DV_IN(channel)

Description:

DV_IN returns the value, known as a Device Variable, from a remote I/O block such as the Flexslice temperature modules, thermocouple and RTD type.  During system startup the channels are allocated according to the DV_MAP, usually there will be 4 or 8 channels per module.

Parameters:

value:

The value read from the device’s input

channel:

0 … max channel number

Examples:

Copy 4 temperature values to local variables, scaling them to degrees Celsius by dividing by 100.  Raw values from the thermocouple slice are degrees x 100 as Integers.

DIM temperature_ch0, temperature_ch1, temperature_ch2,_ temperature_ch3 AS FLOAT

DIM data_index AS INTEGER = 0

 

' read 4 x temperature values from P367

temperature_ch0 = DV_IN(data_index) / 100

temperature_ch1 = DV_IN(data_index + 1) / 100

temperature_ch2 = DV_IN(data_index + 2) / 100

temperature_ch3 = DV_IN(data_index + 3) / 100

 

 

' Compare with setpoints, set output channel to turn on heater

  IF temperature_ch0 < sp1 THEN

      DV_OUT(data_index) = 1

  ELSE

      DV_OUT(data_index) = 0

  ENDIF

 

  IF temperature_ch1 < sp2 THEN

      DV_OUT(data_index + 1) = 1

  ELSE

      DV_OUT(data_index + 1) = 0

  ENDIF

 

  IF temperature_ch2 < sp3 THEN

      DV_OUT(data_index + 2) = 1

  ELSE

      DV_OUT(data_index + 2) = 0

  ENDIF

 

  IF temperature_ch3 < sp4 THEN

      DV_OUT(data_index + 3) = 1

  ELSE

      DV_OUT(data_index + 3) = 0

  ENDIF

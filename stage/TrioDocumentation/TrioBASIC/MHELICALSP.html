MHELICALSP

Type:

Axis Command

Syntax:

MHELICALSP(end1, end2, centre1, centre2, direction, distance3[, mode])

Description:

Performs a helical move as per MHELICAL but with advanced speed control (allows vector speed to be changed when using multiple moves in the buffer), using the axis parameters FORCE_SPEED , ENDMOVE_SPEED . and STARTMOVE_SPEED .

Multi-axis interpolation uses the UNITS, SPEED, ACCEL and DECEL of the BASE axis.  For correct XYZ vector operation, the UNITS of all 3 axes must be the same.

Example:

In a series of buffered moves using the look ahead buffer with MERGE = ON a helical move is required where the incoming vector speed is 40 UNITS /second and the finishing vector speed is 20 UNITS /second.

FORCE_SPEED = 40

ENDMOVE_SPEED = 20

MHELICALSP(100, 100, 0, 100, 1, 100)
